## 4. C组：语法重点

### 4.1 N임을 밝히다

| 项目 | 内容 |
|---|---|
| 核心意思 | 说明“是…… / 会……”这一事实 |
| 例句 | 복제품 전시임을 밝히지 않았다. |
| 中文 | 没有说明这是复制品展览。 |

```korean
추가 비용이 발생할 수 있음을 분명히 밝혔다.
```

> 明确说明了可能会产生追加费用。

---

### 4.2 -지 않고

| 项目 | 内容 |
|---|---|
| 核心意思 | 不……而…… |
| 例句 | 문제를 숨기지 않고 정확히 설명했다. |
| 中文 | 没有隐瞒问题，而是准确说明了。 |

---

### 4.3 -아/어서야 / -고 나서야

| 项目 | 内容 |
|---|---|
| 核心意思 | 到……之后才…… |
| 例句 | 학교에 찾아와서야 사실을 알게 되었다. |
| 中文 | 到了学校之后才知道事实。 |

```korean
수업이 끝나고 나서야 마감일을 알게 되었다.
```

> 课程结束之后才知道截止日期。

---

### 4.4 -게 되다

| 项目 | 内容 |
|---|---|
| 核心意思 | 变得……；最终……；得知…… |

## 5. 增加 / 减少 / 趋势词

### 5.1 增加类

| 韩语 | 中文 |
|---|---|
| 늘다 | 增加、变多 |
| 늘어나다 | 增加、增长 |
| 증가하다 | 增加 |
| 많아지다 | 变多 |
| 오르다 | 上升 |
| 상승하다 | 上升 |
| 높아지다 | 变高 |
| 커지다 | 变大 |
| 급증하다 | 急剧增加、暴增 |

### 5.2 减少类

| 韩语 | 中文 |
|---|---|
| 줄다 | 减少、变少 |
| 줄어들다 | 减少 |
| 감소하다 | 减少 |
| 적어지다 | 变少 |
| 떨어지다 | 下降、掉下 |
| 하락하다 | 下降 |
| 낮아지다 | 变低 |
| 작아지다 | 变小 |
| 급감하다 | 急剧减少、暴减 |

### 5.3 自己变 vs 人为让它变

| 自己变 | 中文 | 人为让它变 | 中文 |
|---|---|---|---|
| 늘다 | 增加 | 늘리다 | 增加、扩大 |
| 줄다 | 减少 | 줄이다 | 减少、缩减 |
| 오르다 | 上升 | 올리다 | 提高 |
| 낮아지다 | 变低 | 낮추다 | 降低 |
| 높아지다 | 变高 | 높이다 | 提高 |
| 커지다 | 变大 | 키우다 | 扩大、培养 |

## 6. 常错词与错因

| 易错项 | 正确意思 | 错因提醒 |
|---|---|---|
| 강의 | 课程、讲课 | 不是讲义；讲义一般是 강의 자료 |
| 단점 | 缺点 | 장점 = 优点 |
| 잦다 | 频繁 | 짧다 = 短 |
| 정부 | 政府 | 정보 = 信息；정거장 = 站台 |
| 편지 | 信 | 문자 = 短信 |
| 제목 | 标题、题目、书名 | 문제 = 问题、题目 |
| 부족하다 | 不足、不够 | 不是“充足” |
| 분명히 | 明确地、清楚地 | 不是“详细地”；详细地 = 자세히 |
| 추가 비용 | 追加费用、额外费用 | 不是“增加的费用” |
| 발생하다 | 发生、产生 | 비용이 발생하다 = 产生费用 |
| 신뢰 | 信任 | 신뢰를 잃다 = 失去信任 |
| 진심이 담긴 | 饱含真心的 | 담기다 = 包含、装着 |
| 한참 | 好一会儿 | 한참 들여다보다 = 盯着看好一会儿 |
| 식습관 | 饮食习惯 | 습관 = 习惯 |
| 눈에 띄게 | 明显地 | 不是“眼睛……” |
