## 7. 今日语法总整理

### 7.1 普通阅读语法

| 语法 | 中文核心 | 例句 |
|---|---|---|
| N임을 밝히다 | 说明“……这一事实” | 환불이 불가능함을 밝혔다. |
| -지 않고 | 不……而…… | 단점을 숨기지 않고 설명했다. |
| -아/어서야 / -고 나서야 | 到……之后才…… | 수업이 끝나고 나서야 알게 되었다. |
| -게 되다 | 变得……；最终……；得知…… | 사실을 알게 되었다. |
| N에 대한 N | 对……的…… | 미술에 대한 관심 |
| -다가 | 做着做着，然后…… | 길을 걷다가 친구를 만났다. |
| -더니 | 做了……之后，结果…… | 물을 주었더니 잎이 풍성해졌다. |
| -다는 건지 | 到底是说……这件事 | 도대체 무슨 말이라는 건지 모르겠다. |
| -아/어 보니 | 做了之后发现…… | 확인해 보니 문제가 있었다. |
| -아/어 있다 | 动作完成后的状态持续 | 쓰러져 있었다. |
| -아/어 놓다 | 做完并保持结果 | 옮겨 놓았다. |
| A-아/어지다 | 变得…… | 풍성해졌다. |
| -나 보다 | 看来、好像 | 필요한가 보다. |
| -(으)려면 | 如果想…… | 잘하려면 알아야 한다. |
| -아/어야 하다 | 必须…… | 제대로 알아야 한다. |
| -기로 하다 | 决定…… | 준비하기로 했다. |

| 语法 | 中文核心 | 例句 |

| -자 | 一……就……；当……后 | 콘서트를 열자 관중이 몰렸다. |
| -면서 | 随着……；一边……一边…… | 금리가 떨어지면서 시장이 살아났다. |
| -기 시작하다 | 开始…… | 시장이 살아나기 시작했다. |
| -지만 | 虽然……但是…… | 하락했지만 거래는 줄어들고 있다. |
