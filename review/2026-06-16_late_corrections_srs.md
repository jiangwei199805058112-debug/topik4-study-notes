## 2. 必须区分：금리 vs 환율

| 韩语 | 中文 | 常见搭配 |
|---|---|---|
| 금리 | 利率 | 대출 금리, 예금 금리, 금리 하락세, 금리 안정화 |
| 환율 | 汇率 | 환율 상승, 환율 하락, 원/달러 환율 |

记忆句：

```korean
대출 금리가 떨어지면서 시장이 살아나기 시작했다.
```

中文：

```text
随着贷款利率下降，市场开始复苏。
```

## 3. 必须区分：찾아오다 vs 찾다

例句：

```korean
모처럼 찾아온 황금연휴에 여행 상품을 찾는 사람이 많아졌다.
```

中文：

```text
难得到来的黄金连休期间，寻找旅游产品的人变多了。
```

| 表达 | 原形 | 本句意思 | 说明 |
|---|---|---|---|
| 찾아온 | 찾아오다 | 到来的 | 假期/机会/春天“来了” |
| 찾는 | 찾다 | 寻找的 | 找商品、找答案、找东西 |

### 记忆例句

| 韩语 | 中文 |
|---|---|
| 봄이 찾아왔다. | 春天来了。 |
| 기회가 찾아왔다. | 机会来了。 |
| 모처럼 찾아온 황금연휴 | 难得到来的黄金连休 |
| 여행 상품을 찾는 사람 | 寻找旅游产品的人 |
| 답을 찾다 | 找答案 |
| 병원을 찾다 | 去医院、前往医院 |
| 전시회를 찾다 | 去参观展览 |

核心记忆：

```text
찾아오다 = 来到、到来
찾다 = 找、寻找；也可以表示访问、前往
```

## 4. 必须区分：모으다 vs 몰리다

| 韩语 | 中文 | 动作方向 | 例句 |
|---|---|---|---|
| 모으다 | 聚集、召集、收集、吸引 | 主体把别人聚过来 | 관중을 모으다 |
| 몰리다 | 聚集、涌来、蜂拥而至 | 人/订单/咨询自己涌来 | 관중이 몰리다 |

### 重点对比

```korean
관중을 모으려고 적극적으로 홍보했다.
```

中文：

```text
为了吸引观众，积极地宣传了。
```

```korean
콘서트를 열자 수많은 관중이 몰렸다.
```

中文：

```text
一举行演唱会，大批观众就涌来了。
```

记忆：

```text
모으다 = 我把人聚过来、吸引过来
몰리다 = 人自己涌过来
```

## 6. 晚间回炉句批改记录

| 큰 기대에도 불구하고 상을 받지 못했다. | 把 상을 받다 理解成“获胜” | 尽管有很高期待，但没能获奖。 |
