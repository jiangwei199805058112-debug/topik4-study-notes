## 2. 必须区分：금리 vs 환율

| 韩语 | 中文 | 常见搭配 |
|---|---|---|
| 금리 | 利率 | 대출 금리, 예금 금리, 금리 하락세, 금리 안정화 |
| 환율 | 汇率 | 환율 상승, 환율 하락, 원/달러 환율 |

记忆句：

```korean
대출 금리가 떨어지면서 시장이 살아나기 시작했다.
```

中文：

```text
随着贷款利率下降，市场开始复苏。
```

## 3. 必须区分：찾아오다 vs 찾다

例句：

```korean
모처럼 찾아온 황금연휴에 여행 상품을 찾는 사람이 많아졌다.
```

中文：

```text
难得到来的黄金连休期间，寻找旅游产品的人变多了。
```

| 表达 | 原形 | 本句意思 | 说明 |
|---|---|---|---|
| 찾아온 | 찾아오다 | 到来的 | 假期/机会/春天“来了” |
| 찾는 | 찾다 | 寻找的 | 找商品、找答案、找东西 |

### 记忆例句

| 韩语 | 中文 |
|---|---|
| 봄이 찾아왔다. | 春天来了。 |
| 기회가 찾아왔다. | 机会来了。 |
| 모처럼 찾아온 황금연휴 | 难得到来的黄金连休 |
| 여행 상품을 찾는 사람 | 寻找旅游产品的人 |
| 답을 찾다 | 找答案 |
| 병원을 찾다 | 去医院、前往医院 |
| 전시회를 찾다 | 去参观展览 |

核心记忆：

```text
찾아오다 = 来到、到来
찾다 = 找、寻找；也可以表示访问、前往
```

## 4. 必须区分：모으다 vs 몰리다

| 韩语 | 中文 | 动作方向 | 例句 |
|---|---|---|---|
| 모으다 | 聚集、召集、收集、吸引 | 主体把别人聚过来 | 관중을 모으다 |
| 몰리다 | 聚集、涌来、蜂拥而至 | 人/订单/咨询自己涌来 | 관중이 몰리다 |

### 重点对比

```korean
관중을 모으려고 적극적으로 홍보했다.
```

中文：

```text
为了吸引观众，积极地宣传了。
```

```korean
콘서트를 열자 수많은 관중이 몰렸다.
```

中文：

```text
一举行演唱会，大批观众就涌来了。
```

记忆：

```text
모으다 = 我把人聚过来、吸引过来
몰리다 = 人自己涌过来
```

## 9. 最终回炉文章新增词与句型

用途：记录最后一轮“下雨、交通、市民不满”新闻式翻译练习中新增或暴露不熟的词。明天可作为第一批回炉。

| 韩语 | 中文 | 记忆点 | 例句 |
|---|---|---|---|
| 시민 | 市民 | 城市里的居民、公众 | 시민들의 불만 |
| 주민 | 居民 | 某地区住的人 | 주민들의 걱정 |
| 불편 | 不便 | 常与 겪다 连用 | 큰 불편을 겪다 |
| 불편을 겪다 | 遇到不便 | 겪다=经历、遭遇 | 출근하던 사람들이 불편을 겪었다. |
| 충분하지 않다 | 不充分、不足 | 충분하다 的否定 | 대책이 충분하지 않다. |
| 교통 대책 | 交通对策 | 교통=交通，대책=对策 | 교통 대책이 필요하다. |
| 기관 | 机构、机关 | 不是“器官”的语境 | 기관에 대해 불만을 쏟아 내다. |
| 반복되다 | 反复发生 | 被动/自动发生 | 문제가 반복되고 있다. |
| 비가 올 때마다 | 每次下雨 | -ㄹ 때마다=每当……时 | 비가 올 때마다 길이 막힌다. |
| 서두르다 | 赶忙、匆忙 | 서둘렀다=赶忙了 | 늦지 않으려고 서둘렀다. |
| 지연되다 | 被延迟、延误 | 운행/일정 常用 | 버스 운행이 지연되었다. |
| 임시 대책 | 临时对策 | 임시=临时 | 임시 대책만으로는 부족하다. |
| -만으로는 | 仅凭……的话 | 后面常接 어렵다/부족하다 | 약속만으로는 신뢰를 얻기 어렵다. |
| 의견 | 意见 | 의견에 대해 답하다 | 시민들의 의견 |
| 의견에 대해 답하다 | 回应意见 | 답하다=回答、回应 | 시민들의 의견에 대해 답하다. |
| 주장하다 | 主张 | -아/어야 한다고 주장하다 | 변화가 나타나야 한다고 주장했다. |
| 나타나다 | 出现 | 변화/문제/효과 常用 | 변화가 나타나다. |
| 나타나야 한다 | 必须出现 | 나타나다 + -아/어야 하다 | 실제 변화가 나타나야 한다. |
| 걱정 | 担心 | 걱정이 커지다=担心增加 | 주민들의 걱정 |
| 걱정이 커지고 있다 | 担心正在增加 | 커지다=变大、增加 | 주민들의 걱정이 커지고 있다. |
| 커지고 있다 | 正在变大、正在增加 | 불만/걱정/관심 常用 | 시민들의 불만이 커지고 있다. |
| 실제 변화 | 实际变化 | 실제=实际 | 실제 변화가 나타나다. |
| 실질적인 대책 | 实际性的对策 | 실질적=实质性的 | 실질적인 대책을 요구하다. |
