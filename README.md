## 语法整理规则

每个语法尽量按这个格式整理：

1. 核心意思
2. 本句意思
3. 常见用法
4. 易混语法对比
5. 例句
6. 我自己的错因

不要只背某一句里的中文翻译，要理解这个语法在句子里的功能。

---

## 单词整理规则

每个单词尽量分三层：

| 项目 | 说明 |
|---|---|
| 核心意思 | 这个词本身最核心的意思 |
| 常用意思 | TOPIK和生活中常见的翻译 |
| 本句意思 | 当前句子里最自然的翻译 |

例如：

| 韩语 | 核心意思 | 常用意思 | 本句意思 |
|---|---|---|---|
| 익히다 | 通过练习掌握 | 学会、掌握、熟练 | 掌握表达 |
| 지나치다 | 经过而不处理 | 经过、错过、跳过 | 跳过不懂的表达 |

## 每日学习流程

1. 做少量真题或句子训练。
2. 把不会的单词和语法扒出来。
3. 只整理高频、影响理解、反复错的内容。
4. 当天背 10~20 个单词、3~5 个语法。
5. 第二天复习昨天的内容。
6. 一周后回看错题和例句。

---

## 当前最重要的原则

不要死背大量语法表。更有效的方法是：

```text
韩文句子 → 反应中文大意 → 判断语法功能 → 回到例句复习
```

推荐比例：

```text
70% 句子反应训练
20% 语法表复习
10% 自己造句
```
