# 固定搭配与短语讲解模块

目的：普通词汇优先拆成单词记忆；只有 TOPIK 高频搭配、广告题关键词、惯用表达、不能按字面理解的表达，才进入本模块。

---

## 1. 收录标准

以下内容可以收录为短语：

1. 高频固定搭配。
2. 广告、公告、阅读题中常整块出现的关键词。
3. 单独拆开后会影响理解的表达。
4. 惯用表达或不能直译的表达。
5. 用户反复错、容易和单词混淆的表达。

普通单词不放进本模块，仍放进 `vocabulary/`。

---

## 2. 每个短语的讲解格式


## 3. 示例：발이 편하다

### 短语

```korean
발이 편하다
```

### 核心意思

脚舒服，穿着不累。

### 逐词拆解

| 部分 | 意思 |
|---|---|
| 발 | 脚 |
| 이 | 主语助词 |
| 편하다 | 舒服、方便 |

### 自然翻译

```text
脚舒服
穿着舒服
走路不累
```

不要翻成“脚方便”。

### 常见场景

鞋、皮鞋、运动鞋、长时间走路、上班通勤。

### TOPIK 判断关键词

```text
오래 걸어도
발이 아프지 않다
가볍다
신기 좋다
구두
운동화
```

### 易混表达

| 表达 | 中文 | 区别 |
|---|---|---|
| 발이 편하다 | 脚舒服 | 鞋类广告常用 |
| 사용하기 편하다 | 使用方便 | 工具、服务类 |
| 이동이 편리하다 | 移动方便 | 包、行李箱、交通类 |

### 例句

```korean
이 구두는 오래 걸어도 발이 편해요.
```

中文：这双皮鞋即使走很久，脚也舒服。

### 记忆钉子

```text
발이 편하다 = 脚舒服 = 鞋类广告
```

## 4. 示例：더러워진 옷

### 短语

```korean
더러워진 옷
```

### 核心意思

变脏的衣服。

### 逐词拆解

| 部分 | 意思 |
|---|---|
| 더럽다 | 脏 |
| 더러워지다 | 变脏 |
| 더러워진 | 变脏的 |
| 옷 | 衣服 |

### 自然翻译

```text
变脏的衣服
弄脏了的衣服
```

只翻成“脏衣服”也能大概理解，但会漏掉 `-아/어지다 = 变得`。

### 常见场景

洗衣店、清洗服务、生活服务广告。

### TOPIK 判断关键词

```text
세탁소
빨다
세탁하다
맡기다
깨끗하게
새 옷처럼
얼룩
두꺼운 이불
```

### 易混表达

| 表达 | 中文 | 区别 |
|---|---|---|
| 더러운 옷 | 脏衣服 | 描述当前状态 |
| 더러워진 옷 | 变脏的衣服 | 强调从干净到脏的变化 |
| 깨끗해진 옷 | 变干净的衣服 | 方向相反 |

### 例句

```korean
더러워진 옷은 세탁소에 맡기세요.
```

中文：变脏的衣服请交给洗衣店。

### 记忆钉子

```text
더러워진 옷 + 세탁소 = 洗衣店广告
```

## 5. 示例：머리를 맞대다

### 短语

```korean
머리를 맞대다
```

### 核心意思

一起商量，一起想办法。

### 逐词拆解

| 部分 | 意思 |
|---|---|
| 머리 | 头 |
| 를 | 宾语助词 |
| 맞대다 | 对着、贴着 |

### 自然翻译

```text
一起商量
一起想办法
共同讨论
```

不能按字面翻成“把头贴在一起”。

### 常见场景

学校、公司、社区、解决问题、寻找办法。

### TOPIK 判断关键词

```text
문제를 해결하다
방법을 찾다
의논하다
함께 생각하다
살릴 방법
```

### 易混表达

| 表达 | 中文 | 区别 |
|---|---|---|
| 머리를 맞대다 | 一起商量 | 惯用表达 |
| 의논하다 | 商量 | 普通动词 |
| 고민하다 | 苦恼、思考 | 不一定一起 |

### 例句

```korean
학생들은 문제를 해결하기 위해 머리를 맞댔다.
```

中文：学生们为了解决问题一起商量。

### 记忆钉子

```text
머리를 맞대다 = 大家凑在一起想办法
```

## 7. 复习方式

固定搭配复习分两层：

1. 看到短语能说出自然中文。
2. 看到场景能判断题目答案。

例如：

```text
발이 편하다 → 脚舒服 → 鞋类广告
더러워진 옷 → 变脏的衣服 → 洗衣店
머리를 맞대다 → 一起商量 → 解决问题
```
