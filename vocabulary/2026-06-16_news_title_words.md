## 9. 语法整理

| 语法 | 核心意思 | 例句 |
|---|---|---|
| -자 | 一……就……；当……后 | 콘서트를 열자 수많은 관중이 몰렸다. |
| -면서 | 随着……；一边……一边…… | 대출 금리가 떨어지면서 시장이 살아났다. |
| -기 시작하다 | 开始…… | 시장이 살아나기 시작했다. |
| -지만 | 虽然……但是…… | 하락했지만 거래는 줄어들고 있다. |
| -으나 | 虽然……但是……；书面 | 콘서트를 열었으나 호응이 적었다. |
| -려고 | 为了…… | 관중을 모으려고 홍보했다. |
| -고자 | 为了……；书面 | 금메달을 따고자 훈련 중이다. |
| -더라도 | 即使……也…… | 값이 비싸더라도 여행을 가려는 사람이 많다. |
| -에도 불구하고 | 尽管……仍然…… | 기대에도 불구하고 상을 받지 못했다. |
| -기로 하다 | 决定…… | 다시 콘서트를 열기로 했다. |
| N에 대한 N | 对……的…… | 원작 작가에 대한 관심 |
| -되다 | 被……；变成…… | 대책이 논의되고 있다. |
| V-(으)ㄴ N | ……的名词 | 갑자기 많이 쏟아진 비 |

## 11. 最容易错的对比

| 容易混 | 正确区分 |
|---|---|
| 수상 유력 | 有望获奖，不是已经获奖 |
| 수상했다 / 상을 받았다 | 已经获奖 |
| 기대 고조 | 期待升高 |
| 기뻐하다 | 高兴，通常是已经有好结果 |
| 기지개 | 市场回暖，不是交易减少 |
| 지지부진하다 | 进展慢、不顺利 |
| 속 터지다 | 急得要命、憋闷 |
| 껑충 | 猛增，不是减少 |
| 원작 | 原作，不是作者 |
| 시청률 | 收视率，不是销量 |
| 판매량 | 销售量 |
| 판매율 | 销售率 |
| 차량 | 车辆 |
| 대중교통 이용자 | 公共交通使用者，和车辆不是一回事 |
