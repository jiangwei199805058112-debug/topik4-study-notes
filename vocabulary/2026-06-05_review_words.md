# 2026-06-05 回炉单词短表

## 今天必须背

1. 제대로 = 好好地、正确地；제대로 익히다 = 正确掌握。
2. 수밖에 없다 = 只能、不得不；실수할 수밖에 없다 = 只能犯错。
3. 느껴지다 = 感觉到、觉得；-처럼 느껴지다 = 感觉像……。
4. 반드시 = 一定、必须；반드시 복습하다 = 一定要复习。
5. 관련되다 = 与……相关；일상생활과 관련되다 = 与日常生活相关。
6. 등장하다 = 出现、登场；문제에 등장하다 = 在题目中出现。
7. 피하다 = 避开、逃避；어려운 문장을 피하다 = 避开难句。
8. 파악하다 = 把握、理解清楚；중심 내용을 파악하다 = 把握中心内容。
9. 속 = 里面、……之中；문장 속 = 句子中；생활 속 = 生活中。
10. 문장 속에서 = 在句子中、在句子语境中。
11. 생활 속 표현 = 生活中的表达。
