## 易混点

| 韩语 | 不要这样理解 | 正确反应 |
|---|---|---|
| 전혀 기억이 나지 않다 | 完全没背下来 | 完全想不起来 |
| 단순히 A가 아니라 B | 单纯地不做A做B | 不只是A，而是B |
| 속 | 只理解成物理里面 | 也可表示抽象范围：문장 속, 생활 속 |
| 제대로 | 只理解成“认真” | 好好地、正确地、做到位 |
| 수밖에 없다 | 有机会 | 只能、不得不 |
| 느껴지다 | 主动感觉 | 被感觉为、感觉起来 |
| 함께 | 单独 | 一起、和……一起 |

## 今日最低背诵清单

1. `제대로` = 好好地、正确地。
2. `수밖에 없다` = 只能、不得不。
3. `느껴지다` = 感觉到、觉得。
4. `반드시` = 一定、必须。
5. `관련되다` = 与……相关。
6. `등장하다` = 出现、登场。
7. `피하다` = 避开、逃避。
8. `파악하다` = 把握、理解清楚。
9. `문장 속에서` = 在句子中。
10. `생활 속 표현` = 生活中的表达。
11. `전혀 기억이 나지 않다` = 完全想不起来。
12. `단순히 A가 아니라 B` = 不只是A，而是B。
13. `-면서` = 一边……一边……。

## 复习例句

```korean
문장 속에서 표현을 익히는 것이 더 효과적이다.
```

中文：在句子中掌握表达更有效。

```korean
단어의 뜻만 외우면 실제 글에서 어떻게 쓰이는지 전혀 기억이 나지 않을 수 있다.
```

中文：如果只背单词意思，在实际文章里可能完全想不起它怎么用。

```korean
중심 내용을 파악하면서 모르는 표현을 함께 정리해야 한다.
```

中文：要一边把握中心内容，一边一起整理不懂的表达。

```korean
단순히 단어를 외우는 것이 아니라 생활 속 표현도 익히는 것이 좋다.
```

中文：不只是背单词，也最好掌握生活中的表达。

```korean
어려운 문장을 계속 피하면 실수할 수밖에 없다.
```

中文：如果一直避开难句，就只能出错。
