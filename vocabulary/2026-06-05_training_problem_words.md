## 最低背诵清单

今天最优先钉住：

1. `어려운` = 难的，不是“男的”。
2. `문장` = 句子。
3. `글` = 文章。
4. `지문` = 阅读原文。
5. `질문` = 问题、提问。
6. `선택지` = 选项。
7. `고르다` = 选择。
8. `답을 찍다` = 蒙答案。
9. `흐름` = 脉络。
10. `쓰임` = 用法。
11. `구별하다` = 区分。
12. `영수증` = 收据、小票。
13. `상황` = 情况、状况。
14. `반복해서` = 反复地。
15. `그래도` = 即使如此、不过。

## 易混对比

| 韩语 | 中文 | 对比 |
|---|---|---|
| 문장 | 句子 | `긴 문장` = 长句 |
| 글 | 文章 | `긴 글` = 长文章 |

| 등장하다 | 出现、登场 | `글에 등장하다` |
| 찍다 | 蒙答案 | 考试语境 |
| 풀다 | 做题、解题 | `문제를 풀다` |
