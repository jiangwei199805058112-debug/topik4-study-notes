# 单词整理粒度规则

目的：整理 TOPIK 单词时，优先记录“单词本身”，不要把所有内容都做成长短语，避免背诵负担过重。

---

## 1. 总原则

以后整理词汇时：

```text
优先整理单词。
只有高频、固定搭配、题目判断关键词，才整理为短语。
```

---

## 2. 优先作为单词整理

以下内容尽量拆成单词记忆：

| 韩语 | 中文 |
|---|---|
| 가볍다 | 轻 |
| 무겁다 | 重 |
| 더럽다 | 脏 |
| 피하다 | 避开、避免 |
| 피해 | 损害、受害 |
| 비율 | 比率 |
| 제출하다 | 提交 |
| 내려받다 | 下载 |
| 긴장되다 | 紧张 |
| 통과하다 | 通过、穿过 |
| 스며들다 | 渗入、渗透 |

---

## 3. 可以作为高频短语整理

以下情况保留短语，因为 TOPIK 中经常整块出现，或者单独拆开会影响做题：

| 韩语 | 中文 | 保留原因 |
|---|---|---|
| 발이 편하다 | 脚舒服 | 广告题高频关键词 |
| 더러워진 옷 | 变脏的衣服 | 洗衣店广告题常见组合 |
| 두꺼운 이불 | 厚被子 | 洗衣店/生活服务题高频 |
| 무거운 짐 | 重行李 | 行李箱/包广告题关键词 |
| 이동이 편리하다 | 移动方便 | 广告题固定表达 |
| 배터리 교환 | 更换电池 | 修理店广告题常见组合 |
| 열이 내리다 | 退烧 | 医疗/健康常见表达 |
| 시간을 미루다 | 推迟时间 | 约定/会议常见表达 |
| 신청서를 제출하다 | 提交申请书 | 公告/报名题高频 |
| 머리를 맞대다 | 一起商量、一起想办法 | 惯用表达，不能按字面理解 |
| 그대로 따르다 | 原样照着做 | 阅读题高频表达 |

---

## 4. 拆分方式

遇到短语时，优先这样整理：

```text
核心单词 + 高频搭配
```

例如：

| 核心单词 | 高频搭配 |
|---|---|
| 더럽다 / 더러워지다 | 더러워진 옷 |
| 가볍다 | 가벼운 가방, 가벼운 신발 |
| 피하다 | 위험을 피하다 |
| 피해 | 피해를 막다 |
| 제출하다 | 신청서를 제출하다 |
| 미루다 | 약속 시간을 미루다 |

---

## 5. 后续词表格式

后续词表默认格式：

| 序号 | 韩语 | 词性 | 中文释义 | 例句 |
|---:|---|---|---|---|
| 1 | 가볍다 | 形容词 | 轻 | 이 가방은 가벼워요. |

如果是高频短语，则在词性栏写“短语”或“惯用表达”：

| 序号 | 韩语 | 词性 | 中文释义 | 例句 |
|---:|---|---|---|---|
| 1 | 머리를 맞대다 | 惯用表达 | 一起商量、一起想办法 | 문제를 해결하기 위해 머리를 맞댔다. |

---

## 6. 执行要求

1. 不把所有句子都整理成短语。
2. 普通词优先拆成单词。
3. 高频固定搭配、广告题关键词、惯用表达可以保留为短语。
4. 同一个短语如果包含易混单词，要同时列出核心单词。
5. 复习时优先测单词，短语只测高频搭配。
