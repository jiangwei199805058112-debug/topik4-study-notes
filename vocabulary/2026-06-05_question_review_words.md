## 最低背诵清单

今天最优先背这些：

1. `발이 편하다` = 脚舒服。
2. `가볍다` = 轻。
3. `무거운 짐` = 重行李。
4. `옮기다` = 搬、移动。
5. `바퀴` = 轮子。
6. `더러워진 옷` = 变脏的衣服。
7. `두꺼운 이불` = 厚被子。
8. `활기찬` = 充满活力的。
9. `달리기` = 跑步。
10. `집 앞까지 배달하다` = 配送到家门口。
11. `고장 난 시계` = 坏了的手表。
12. `배터리 교환` = 更换电池。
13. `신청서를 작성하다` = 填写申请表。
14. `제출 버튼을 누르다` = 按提交按钮。
15. `등록이 완료되다` = 登记完成。
16. `예매 방법` = 预订方法。
17. `지문 / 질문 / 선택지` = 原文 / 问题 / 选项。
18. `답을 고르다` = 选择答案。
19. `답을 찍다` = 蒙答案。
20. `흐름을 놓치다` = 没抓住脉络。

## 广告/说明题判断公式

| 关键词 | 判断对象 |
|---|---|
| `발이 편하다`, `가볍다`, `디자인` | 구두 / 신발 |
| `더러워진 옷`, `두꺼운 이불`, `맡기다` | 세탁소 |
| `달리기`, `활기찬 내일` | 건강 관리 |
| `공연 날짜`, `인원`, `좌석`, `결제` | 예매 방법 |
| `비 오는 날`, `튼튼하다`, `들고 다니다` | 우산 |
| `시원한 바람`, `더위`, `전기료` | 선풍기 |
| `신청서`, `제출`, `등록 완료` | 등록 방법 |
| `무거운 짐`, `바퀴`, `이동이 편리하다` | 가방 |
| `신선한 우유`, `집 앞까지`, `배달` | 배달 서비스 |
| `고장 난 시계`, `고치다`, `배터리 교환` | 시계 수리점 |
| `책`, `귀로 듣다` | 오디오북 서비스 |
