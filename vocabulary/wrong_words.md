# 常错单词与高频词整理

整理规则：每个词尽量同时记录核心意思、常用意思、本句意思、常见搭配和例句。不要只背当前句子里的中文翻译。

## 重点区分

### 배우다 / 외우다 / 익히다

| 韩语 | 核心意思 | 重点 |
|---|---|---|
| 배우다 | 学习 | 开始学、接受知识 |
| 외우다 | 背、记住 | 记忆内容 |
| 익히다 | 掌握、练熟 | 学到能用 |

例句：

```korean
문법을 배웠지만 아직 익히지 못했어요.
```

中文：语法学过了，但还没掌握。

```korean
단어만 외우지 말고 예문도 같이 보세요.
```

中文：不要只背单词，也要一起看例句。

## 单词掌握标准

一个词只有做到下面三点，才算真正掌握：

1. 看到韩语能说出核心意思。
2. 能认出常见搭配。
3. 能看懂真题句子里的本句意思。
