# 第一周核心语法整理

本文件整理当前最需要优先掌握的 TOPIK 3~4级基础语法。重点不是死背中文，而是理解语法在句子里的功能。

---

## 1. 经历类：`-(으)ㄴ 적이 있다`

### 核心意思

有过……经历；曾经……过。

### 结构

```text
动词过去修饰形 + 적이 있다/없다
```

### 例句

```korean
저는 어릴 때 피아노를 배운 적이 있어요.
```

中文：我小时候学过钢琴。

```korean
이 식당에 와 본 적이 없어서 길을 잘 몰라요.
```

中文：因为没来过这家餐厅，所以不太认识路。

```korean
그 사람을 직접 만난 적은 없지만 이름은 들어 봤어요.
```

中文：虽然没亲自见过那个人，但听过他的名字。

### 易错点

- `적이 있다` = 有经历。
- `적이 없다` = 没有经历。
- `-아/어 보다` 和 `적이 있다` 经常一起出现：`가 본 적이 있다`, `먹어 본 적이 있다`, `풀어 본 적이 있다`。

---

## 2. 顺序类：`-고 나서`

### 核心意思

做完A之后，再做B。

### 结构

```text
动词词干 + 고 나서
```

### 例句

```korean
설명을 듣고 나서 문제를 다시 풀었어요.
```

中文：听完说明以后，又做了一遍题。

```korean
자료를 확인하고 나서 보고서를 썼어요.
```

中文：确认资料以后，写了报告。

```korean
운동을 하고 나서 샤워를 했어요.
```

中文：运动以后洗了澡。

---

## 3. 顺序强调：`-고 나서야`

### 核心意思

做完A之后，才B。

### 和 `-고 나서` 的区别

| 语法 | 核心意思 | 语气 |
|---|---|---|
| `-고 나서` | 做完A之后B | 普通顺序 |
| `-고 나서야` | 做完A之后才B | 强调之前不行，A之后才发生 |

### 例句

```korean
선생님의 설명을 듣고 나서야 문법을 이해했어요.
```

中文：听完老师的讲解以后，才理解了这个语法。

```korean
직접 해 보고 나서야 어렵다는 것을 알았어요.
```

中文：亲自做过以后，才知道很难。

---

## 4. 条件类：`-거든`

### 核心意思

如果……的话。

### 常见特点

后面常接命令、建议、请求。

### 例句

```korean
피곤하거든 오늘은 일찍 쉬세요.
```

中文：如果累的话，今天早点休息。

```korean
약속 시간이 바뀌거든 미리 알려 주세요.
```

中文：如果约定时间变了，请提前告诉我。

```korean
궁금한 점이 있거든 언제든지 물어보세요.
```

中文：如果有疑问，随时问。

### 易错点

`있거든` 不是过去式。

```text
있다 → 있- + 거든 = 있거든
```

过去式要看到 `었`：

```text
있었거든 = 如果当时有 / 如果已经有过
```

---

## 5. 条件类：`-(으)려면`

### 核心意思

如果想……，就要……

### 结构

```text
动词词干 + (으)려면
```

### 例句

```korean
발음을 고치려면 많이 듣고 따라 해야 해요.
```

中文：如果想纠正发音，就要多听并跟读。

```korean
긴 글을 이해하려면 문단의 중심 내용을 찾아야 해요.
```

中文：如果想理解长文，就要找到段落的中心内容。

```korean
한국 친구와 자연스럽게 대화하려면 표현을 많이 익혀야 해요.
```

中文：如果想和韩国朋友自然对话，就要掌握很多表达。

---

## 6. 必要条件：`-아/어야 하다`

### 核心意思

必须……；要……。

### 结构

```text
动词/形容词 아/어야 하다
```

### 例句

```korean
표현을 많이 익혀야 해요.
```

中文：必须掌握很多表达。

```korean
좋은 점수를 받으려면 실수를 줄여야 해요.
```

中文：如果想取得好分数，就要减少失误。

```korean
문법을 기억하려면 예문과 같이 외워야 해요.
```

中文：如果想记住语法，就要和例句一起背。

### 易错点：`야` 不一定都表示“才”

| 形式 | 语法整体 | 中文核心 |
|---|---|---|
| `-고 나서야` | 做完A之后才B | 才 |
| `-아/어야 하다` | 必须A | 必须、要 |
| `-아/어야 + 결과` | 只有A才B | 只有……才…… |

---

## 7. 目的类：`-고자`

### 核心意思

为了……；想要……。书面语。

### 例句

```korean
부모님께 좋은 소식을 전하고자 열심히 준비했어요.
```

中文：为了把好消息告诉父母，我认真准备了。

```korean
실력을 확인하고자 모의고사를 봤어요.
```

中文：为了确认实力，做了模拟考试。

### 同义表达

```text
-고자 ≈ -기 위해서
```

---

## 8. 目的类：`-기 위해서`

### 核心意思

为了……

### 例句

```korean
생활비를 아끼기 위해서 집에서 밥을 해 먹어요.
```

中文：为了节省生活费，在家做饭吃。

```korean
건강을 회복하기 위해서 잠을 충분히 자려고 해요.
```

中文：为了恢复健康，打算睡够觉。

---

## 9. 排除/替代表达：`-지 말고`

### 核心意思

不要A，而要B。

### 结构

```text
动词词干 + 지 말고 + B
```

### 例句

```korean
모르는 표현을 그냥 지나치지 말고 따로 정리하세요.
```

中文：不要直接跳过不懂的表达，要单独整理。

```korean
답을 바로 보지 말고 먼저 스스로 생각해 보세요.
```

中文：不要马上看答案，先自己想一想。

```korean
어려운 문제만 붙잡고 있지 말고 쉬운 문제부터 푸세요.
```

中文：不要一直抓着难题不放，请从简单题开始做。

---

## 10. 判断/总结类：`-(으)ㄴ 셈이다`

### 核心意思

算是……；等于……；结果上接近……

### 例句

```korean
두 시간 동안 한 문제만 풀었으니 시간이 많이 걸린 셈이에요.
```

中文：两个小时只做了一道题，所以算是花了很多时间。

```korean
설명은 들었지만 기억이 안 나니 다시 배워야 하는 셈이에요.
```

中文：虽然听了讲解，但想不起来，所以算是需要重新学。

```korean
돈을 냈지만 서비스를 받지 못했으니 손해를 본 셈이에요.
```

中文：虽然付了钱，但没得到服务，所以算是吃亏了。

---

## 11. 判断/总结类：`-거나 마찬가지이다`

### 核心意思

和……差不多；几乎等于……

### 说明

这里的 `거나` 不要按普通“或者”理解，要把整个句型当成固定表达：

```text
V-(으)ㄴ 거나 마찬가지이다
= 和做了/没做某事差不多
```

### 例句

```korean
수업에 들어갔지만 계속 휴대폰만 봤으니 안 들은 거나 마찬가지예요.
```

中文：虽然进了课堂，但一直只看手机，所以和没听课差不多。

```korean
단어장을 만들었지만 한 번도 보지 않았으니 안 만든 거나 마찬가지예요.
```

中文：虽然做了单词本，但一次都没看，所以和没做差不多。

```korean
문장을 읽었지만 뜻을 전혀 몰랐으니 안 읽은 거나 마찬가지예요.
```

中文：虽然读了句子，但完全不知道意思，所以和没读差不多。

# 本周新增：TOPIK 基础语法题补充

下面这些来自后续前几题/广告题训练，继续归入第一周基础语法池。

---

## 12. 时间经过：`-(으)ㄴ 지`

### 核心意思

做某事以来，已经过了多久。

### 常见结构

```text
动词过去修饰形 + 지 + 时间 + 되다
```

### 例句

```korean
이 동네로 이사를 온 지 일 년이 됐다.
```

中文：搬到这个小区已经一年了。

```korean
한국어를 공부한 지 두 달이 됐어요.
```

中文：学习韩语已经两个月了。

### 易错点

`온 지` 是“来了以后经过多久”，不是“来的时候”。

---

## 13. 时间点：`-(으)ㄹ 때`

### 核心意思

……的时候。

### 例句

```korean
걸을 때 발이 편해요.
```

中文：走路的时候脚很舒服。

```korean
학교에 올 때 우산을 가져오세요.
```

中文：来学校的时候请带伞。

---

## 14. 选择/列举：`-거나`

### 核心意思

或者……；做A或做B。

### 例句

```korean
친구가 오거나 내가 갈 거예요.
```

中文：朋友来，或者我去。

```korean
시간이 있으면 읽거나 듣기 연습을 하세요.
```

中文：有时间的话，做阅读或听力练习。

### 易错点

它和 `-거나 마찬가지이다` 里的 `거나` 不一样；后者要整体背成“和……差不多”。

---

## 15. 过程中发生：`-다가`

### 核心意思

做A的过程中，发生B。常见于中途发生变化、意外或不好的结果。

### 例句

```korean
휴대 전화를 보다가 내려야 할 역을 지나쳤다.
```

中文：看手机的时候，错过了应该下车的站。

```korean
학교에 오다가 친구를 만났어요.
```

中文：来学校的路上遇到了朋友。

```korean
길을 걷다가 넘어졌어요.
```

中文：走路的时候摔倒了。

---

## 16. 逐渐变化：`-아/어 가다`

### 核心意思

逐渐……下去；慢慢变得……

### 例句

```korean
가을이 되면서 나뭇잎 색이 점점 붉게 변해 간다.
```

中文：随着秋天到来，树叶颜色渐渐变红。

```korean
한국어 실력이 조금씩 좋아져 가고 있어요.
```

中文：韩语实力正在一点点变好。

### 易错点

`점점`, `조금씩` 和 `-아/어 가다` 经常一起出现，表示变化过程。

---


---

## 20. 可能：`-(으)ㄹ지도 모르다`

### 核心意思

也许……；可能……

### 例句

```korean

```

中文：现在不出发的话，可能会迟到。

```korean

```

中文：明天也许会下雨。

---

## 21. 也可能：`-(으)ㄹ 수도 있다`

### 核心意思

也可能……；有可能……

### 例句

```korean
늦을 수도 있다.
```

中文：也可能会迟到。

```korean
시간이 부족하면 틀릴 수도 있어요.
```

中文：时间不够的话，也可能会错。

### 易混对比

| 语法 | 核心意思 |
|---|---|
| `-(으)ㄹ지도 모르다` | 也许…… |
| `-(으)ㄹ 수도 있다` | 也可能…… |
| `-(으)ㄹ 리가 없다` | 不可能…… |

---

## 25. 原因负面：`-는/은 탓에`

### 核心意思

因为……导致坏结果；都怪……

### 例句

```korean
비가 오는 탓에 늦었어요.
```

中文：因为下雨，所以迟到了。

```korean
잠을 못 잔 탓에 피곤해요.
```

中文：因为没睡好，所以很累。

### 易错点

它只强调负面原因，不等于“每次……都”。
