# 2026-06-03 句子训练暴露语法

本文件整理本轮“单词 + 语法合成句子训练”中暴露出来的不熟语法。重点放在能不能看到句子后马上反应语法功能。

---

## 1. `-면서`

### 核心意思

1. 一边A一边B。
2. 随着A，B也发生变化。

### 例句

```korean
날씨가 따뜻해지면서 공원에 사람이 많아졌어요.
```

中文：随着天气变暖，公园里的人变多了。

```korean
음악을 들으면서 공부해요.
```

中文：一边听音乐一边学习。

### 易错点

`-면서` 不一定都翻译成“一边……一边……”。如果前后是变化关系，常翻译为“随着……”。

---

## 2. `-(으)ㄹ 수도 있다`

### 核心意思

也可能……；有可能……

### 例句

```korean
시험 날짜가 바뀔 수도 있으니까 안내문을 다시 확인하세요.
```

中文：考试日期也可能会变，所以请重新确认说明。

```korean
시간이 부족하면 틀릴 수도 있어요.
```

中文：时间不够的话，也可能会错。

### 易混对比

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-(으)ㄹ 수도 있다` | 也可能…… | 시험 날짜가 바뀔 수도 있다. |
| `-(으)ㄹ지도 모르다` | 也许…… | 시험 날짜가 바뀔지도 모른다. |
| `-(으)ㄹ 리가 없다` | 不可能…… | 시험 날짜가 갑자기 바뀔 리가 없다. |

---

## 3. `-(으)ㄹ 리가 없다`

### 核心意思

不可能……；没有……的道理。

### 例句

```korean
그 사람이 약속을 잊었을 리가 없어요.
```

中文：那个人不可能忘记约定。

```korean
이렇게 쉬운 문제가 틀릴 리가 없어요.
```

中文：这么简单的题不可能错。

---

## 4. `-다가`

### 核心意思

做A的过程中，发生B。常见于中途发生变化、意外、遗漏或错误。

### 例句

```korean
문제를 풀다가 중요한 단어를 놓쳤어요.
```

中文：做题过程中漏掉了重要单词。

```korean
휴대 전화를 보다가 내려야 할 역을 지나쳤어요.
```

中文：看手机的时候坐过站了。

```korean
길을 걷다가 넘어졌어요.
```

中文：走路的时候摔倒了。

---

## 5. `-기만 하면`

### 核心意思

只要一A，就B。

### 例句

```korean
저는 커피를 마시기만 하면 잠이 잘 안 와요.
```

中文：我只要一喝咖啡，就睡不着。

```korean
이 문제를 보기만 하면 헷갈려요.
```

中文：我只要一看到这个题就混乱。

### 易错点

不是“只喝咖啡”，而是“只要一喝咖啡，就出现某个结果”。

---

## 6. `-(으)ㄹ 때마다`

### 核心意思

每次……的时候都。

### 例句

```korean
지하철을 탈 때마다 사람이 너무 많아서 힘들어요.
```

中文：每次坐地铁时，人都太多，所以很累。

```korean
차를 탈 때마다 멀미를 해요.
```

中文：每次坐车都晕车。

---

## 7. `-는/은 탓에`

### 核心意思

因为A导致坏结果；带“都怪A”的负面感。

### 例句

```korean
비가 오는 탓에 농산물 가격이 올랐어요.
```

中文：因为下雨，农产品价格上涨了。

```korean
잠을 못 잔 탓에 하루 종일 피곤했어요.
```

中文：因为没睡好，一整天都很累。

---

## 8. `-는 바람에`

### 核心意思

因为意外A，导致B。常带突然、倒霉感。

### 例句

```korean
버스를 놓치는 바람에 시험 시간에 늦었어요.
```

中文：因为错过公交，考试迟到了。

```korean
파일을 잘못 지우는 바람에 다시 만들어야 했어요.
```

中文：因为误删文件，所以不得不重新做。

---

## 9. `-다고 해도`

### 核心意思

就算……也……；即使……也……

### 例句

```korean
구두가 예쁘다고 해도 발이 아프면 오래 신기 어려워요.
```

中文：就算皮鞋漂亮，如果脚疼，也很难穿很久。

```korean
다시 설명을 듣는다고 해도 바로 이해하기는 어려울 것 같아요.
```

中文：就算再听一次说明，也好像很难马上理解。

---

## 10. `-거나 마찬가지이다`

### 核心意思

和……差不多；几乎等于……

### 例句

```korean
단어장을 만들었지만 한 번도 보지 않았으니 안 만든 거나 마찬가지예요.
```

中文：虽然做了单词本，但一次都没看，所以和没做差不多。

```korean
문장을 읽었지만 뜻을 전혀 몰랐으니 안 읽은 거나 마찬가지예요.
```

中文：虽然读了句子，但完全不知道意思，所以和没读差不多。

---

## 本轮优先复习顺序

1. `-(으)ㄹ 수도 있다`：也可能……
2. `-(으)ㄹ 리가 없다`：不可能……
3. `-면서`：随着…… / 一边……一边
4. `-다가`：做A过程中发生B
5. `-기만 하면`：只要一……就……
6. `-(으)ㄹ 때마다`：每次……都
7. `-는/은 탓에`：因为……导致坏结果
8. `-는 바람에`：因为意外A导致B
9. `-다고 해도`：就算……也……
10. `-거나 마찬가지이다`：和……差不多
