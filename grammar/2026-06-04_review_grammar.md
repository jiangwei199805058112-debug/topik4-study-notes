# 2026-06-04 回炉语法表

来源：混合翻译训练中卡住的句子。目标不是背中文，而是看到句子时能立刻判断语法功能。

---

## 1. `-아/어야 하다`

### 核心意思

必须……；应该……；要……。

### 本轮句子

```korean
성적을 올리기 위해서 모의고사를 자주 봐야 해요.
```

中文：为了提高成绩，应该经常做模拟考试。

### 拆解

```text
보다 → 봐야 하다 → 봐야 해요
```

`모의고사를 보다` 在考试语境里不是“看模拟考试”，而是“参加/做模拟考试”。

### 例句

```korean
좋은 점수를 받으려면 오답을 다시 정리해야 해요.
```

中文：如果想取得好分数，就要重新整理错题。

---

## 2. `-지 말고`

### 核心意思

不要A，而要B。

### 本轮句子

```korean
그냥 지나치지 말고 모르는 표현은 따로 적어 두세요.
```

中文：不要直接跳过，不懂的表达要单独记下来。

### 拆解

```text
지나치다 → 지나치지 말고
```

`말고` 来自 `말다 + 고`，表示“不要做前面的事，而做后面的事”。

### 例句

```korean
답을 바로 보지 말고 먼저 스스로 생각해 보세요.
```

中文：不要马上看答案，先自己想一想。

---

## 3. `V-(으)ㄴ 지 얼마 안 되다`

### 核心意思

做某事以来没多久；刚……没多久。

### 本轮句子

```korean
세탁소에 옷을 맡긴 지 얼마 안 됐어요.
```

中文：把衣服交给洗衣店还没多久。

### 拆解

```text
맡기다 → 맡긴 지 얼마 안 되다
```

注意：`맡기다` 是“交给别人处理”，不是“已经洗好”。

### 例句

```korean
한국어 공부를 시작한 지 얼마 안 됐어요.
```

中文：开始学习韩语没多久。

---

## 4. `-고자`

### 核心意思

为了……；想要……。偏书面语。

### 本轮句子

```korean
발음을 고치고자 매일 듣고 따라 하고 있어요.
```

中文：为了纠正发音，每天都在听并跟读。

### 同义表达

```text
발음을 고치고자 = 발음을 고치기 위해서
```

### 例句

```korean
문제의 원인을 찾고자 여러 자료를 조사했어요.
```

中文：为了找出问题的原因，调查了很多资料。

---

## 5. `-으니 / -니`

### 核心意思

因为……；既然……。常用于根据前句做判断、建议或结论。

### 本轮句子

```korean
이 표현은 시험에 자주 나오니 꼭 따로 정리하세요.
```

中文：这个表达在考试中经常出现，所以一定要单独整理。

### 例句

```korean
시간이 없으니 중요한 문제부터 풀어야 해요.
```

中文：因为没时间，所以要先做重要的题。

---

## 6. `-(으)ㄹ 리가 없다`

### 核心意思

不可能……。

### 本轮句子

```korean
그렇게 많이 연습했는데 발음이 안 좋아질 리가 없어요.
```

中文：练习了那么多，发音不可能没有变好。

```korean
가격이 이렇게 많이 떨어졌는데 손해를 안 봤을 리가 없어요.
```

中文：价格降了这么多，不可能没亏。

### 易错点

`리가 없다` 不是“没机会”，而是“没有那种可能性/道理”。

---

## 7. `-(으)ㄴ 셈이다` / `N인 셈이다`

### 核心意思

算是……；相当于……。

### 本轮句子

```korean
기본 표현은 익힌 셈이에요.
```

中文：算是掌握了基本表达。

```korean
좋은 습관을 만든 셈이에요.
```

中文：算是养成了好习惯。

### 例句

```korean
매일 예문을 봤으니 문법을 복습한 셈이에요.
```

中文：因为每天看例句，所以算是复习了语法。

---

## 8. `-거나 마찬가지이다`

### 核心意思

和……差不多；几乎就是……。

### 本轮句子

```korean
안 배운 거나 마찬가지예요.
```

中文：和没学差不多。

### 例句

```korean
수업에 들어갔지만 계속 휴대폰만 봤으니 안 들은 거나 마찬가지예요.
```

中文：虽然进了课堂，但一直只看手机，所以和没听课差不多。

## 今日优先复习顺序

| 优先级 | 语法 | 必须反应出来的中文 |
|---|---|---|
| A | `-(으)ㄹ 리가 없다` | 不可能…… |
| A | `-아/어야 하다` | 必须/应该…… |
| A | `-지 말고` | 不要A，而要B |
| A | `V-(으)ㄴ 지 얼마 안 되다` | 做某事没多久 |
| B | `-고자` | 为了…… |
| B | `-으니 / -니` | 因为/既然…… |
| B | `-(으)ㄴ 셈이다` | 算是…… |
| B | `-거나 마찬가지이다` | 和……差不多 |
