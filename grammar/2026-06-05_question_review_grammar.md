# 2026-06-05 题目回归训练语法整理

来源：TOPIK 阅读题回归训练。

重点整理基础语法题和阅读题选项中出现的高频语法。

---

## 1. `V-(으)ㄴ 지 + 时间 + 되다`

### 核心意思

做某事以来已经多久。

### 例句

```korean
이 동네로 이사를 온 지 일 년이 됐다.
```

中文：搬到这个小区已经一年了。

```korean
세탁소에 옷을 맡긴 지 얼마 안 됐어요.
```

中文：把衣服交给洗衣店还没多久。

### 拆解

```text
오다 → 온 지
맡기다 → 맡긴 지
```

### 易错点

`지` 在这里不是 `지문` 的 `지`，而是表示“从做某事到现在经过的时间”。

---

## 2. `A/V-(으)ㄴ/는 N`

### 核心意思

把形容词或动词变成修饰名词的形式：……的N。

### 本轮重点变化

| 原形 | 变化 | 中文 |
|---|---|---|
| 무겁다 | 무거운 짐 | 重行李 |
| 두껍다 | 두꺼운 이불 | 厚被子 |
| 더러워지다 | 더러워진 옷 | 变脏的衣服 |
| 고장 나다 | 고장 난 시계 | 坏了的手表 |
| 원하다 | 원하는 좌석 | 想要的座位 |
| 활기차다 | 활기찬 생활 | 充满活力的生活 |
| 맞다 | 맞는 답 | 正确答案 |

### 例句

```korean
무거운 짐도 쉽게 옮길 수 있어요.
```

中文：重行李也能轻松搬动。

```korean
고장 난 시계는 수리점에 맡기세요.
```

中文：坏了的手表请交给修理店。

```korean
원하는 좌석을 선택하고 결제하세요.
```

中文：请选择想要的座位并付款。

### 易错点

看到 `고장 난`, `더러워진`, `원하는`, `맞는` 时，要先反应它们都在修饰后面的名词。

## 3. `V-아/어야 하다`

### 核心意思

必须……；应该……；得……。

### 例句

```korean
신청서를 작성해야 해요.
```

中文：必须填写申请表。

```korean
제출 버튼을 눌러야 등록이 완료돼요.
```

中文：必须按提交按钮，登记才会完成。

```korean
선택지만 보지 말고 지문도 다시 읽어야 해요.
```

中文：不要只看选项，原文也要重新读。

### 易错点

`-아/어야 하다` 是义务或必要，不是“可以”。看到 `해야 해요`, `읽어야 해요`, `눌러야 해요` 先反应成“必须、得”。

---

## 4. `V-(으)ㄹ 수 있다`

### 核心意思

能……；可以……；有能力或可能性。

### 例句

```korean
바퀴가 있어서 무거운 짐도 쉽게 옮길 수 있어요.
```

中文：因为有轮子，重行李也能轻松移动。

```korean
질문의 뜻을 이해하면 맞는 답을 고를 수 있어요.
```

中文：如果理解问题的意思，就能选出正确答案。

```korean
글의 흐름을 파악하면 어려운 문제도 풀 수 있어요.
```

中文：如果把握文章脉络，难题也能做出来。

### 易错点

阅读题里 `풀 수 있다` 常常是“能做出来、能解题”，不要只按普通“能解决”理解。

---

## 5. `V-(으)면`

### 核心意思

如果……；一旦……。

### 例句

```korean
글의 흐름을 놓치면 맞는 답을 고르기 어려워요.
```

中文：如果漏掉文章脉络，就很难选出正确答案。

```korean
선택지만 보면 답을 잘못 고를 수 있어요.
```

中文：如果只看选项，可能会选错答案。

```korean
제출 버튼을 누르면 등록이 완료돼요.
```

中文：按提交按钮后，登记就完成。

### 易错点

`-(으)면` 不只表示假设，也常用于说明题里的操作条件：“按下后、选择后、提交后”。

---

## 6. `N만` / `N도`

### 核心意思

`만` = 只、仅仅。

`도` = 也。

### 例句

```korean
선택지만 보고 답을 찍으면 안 돼요.
```

中文：不能只看选项就蒙答案。

```korean
질문만 보지 말고 지문도 다시 읽어야 해요.
```

中文：不要只看问题，原文也要重新读。

```korean
무거운 짐도 쉽게 옮길 수 있어요.
```

中文：重行李也能轻松移动。

### 易错点

`만` 会缩小范围，`도` 会扩大或补充范围。阅读题里最常见的错法是只看选项，不回到原文。

---

## 7. `V-고` / `A-고`

### 核心意思

连接两个动作、状态或特征：……并且……；……然后……。

### 例句

```korean
이 구두는 가볍고 디자인도 예뻐요.
```

中文：这双鞋很轻，设计也漂亮。

```korean
공연 날짜와 인원을 선택하고 다음 버튼을 누르세요.
```

中文：请选择演出日期和人数，然后按下一步按钮。

```korean
원하는 좌석을 선택하고 결제하세요.
```

中文：请选择想要的座位并付款。

### 易错点

`-고` 可能是并列，也可能是动作顺序。广告题里常并列优点，说明题里常表示步骤顺序。

---

## 8. `V-면서`

### 核心意思

一边……一边……；两个动作同时进行。

### 例句

```korean
표현의 쓰임을 확인하면서 읽어야 해요.
```

中文：要一边确认表达用法，一边读。

```korean
차례대로 읽으면서 글의 흐름을 파악하세요.
```

中文：请一边按顺序读，一边把握文章脉络。

### 易错点

`-면서` 的重点不是死背“一边”，而是判断前后两个动作是否同时进行。

---

## 9. `V-지 말고`

### 核心意思

不要A，而要B。

### 例句

```korean
선택지만 보지 말고 지문도 다시 읽어야 해요.
```

中文：不要只看选项，原文也要重新读。

```korean
바로 답을 찍지 말고 질문의 뜻을 먼저 확인하세요.
```

中文：不要马上蒙答案，先确认问题的意思。

### 易错点

`-지 말고` 后半句才是应该做的动作。做阅读题时要把注意力放在后面的建议上。

---

## 10. `N처럼`

### 核心意思

像N一样。

### 例句

```korean
더러워진 옷도 새 옷처럼 깨끗하게 해 드립니다.
```

中文：变脏的衣服也会为您洗得像新衣服一样干净。

```korean
이 표현은 광고 문구처럼 자주 나와요.
```

中文：这个表达像广告语一样经常出现。

### 易错点

`처럼` 前面接名词，表示比喻或状态相似。广告题里常用来强调服务效果。

## 今日优先反应顺序

| 优先级 | 语法/变化形 | 必须反应出来的中文 |
|---|---|---|
| A | `V-(으)ㄴ 지 + 时间 + 되다` | 做某事以来已经多久 |
| A | `A/V-(으)ㄴ/는 N` | ……的N |
| A | `V-아/어야 하다` | 必须、应该、得 |
| A | `V-(으)ㄹ 수 있다` | 能、可以 |
| A | `V-(으)면` | 如果…… / 一旦…… |
| B | `N만` | 只、仅仅 |
| B | `N도` | 也 |
| B | `V-고 / A-고` | 并且 / 然后 |
| B | `V-면서` | 一边……一边…… |
| B | `V-지 말고` | 不要A，而要B |
| B | `N처럼` | 像N一样 |

## 题目句型反应训练

1. `이사를 온 지 일 년이 됐다.` = 搬来已经一年了。
2. `고장 난 시계` = 坏了的手表。
3. `더러워진 옷` = 变脏的衣服。
4. `무거운 짐도 옮길 수 있다.` = 重行李也能搬动。
5. `제출 버튼을 누르면 등록이 완료된다.` = 按提交按钮后登记完成。
6. `선택지만 보지 말고 지문도 읽어야 한다.` = 不要只看选项，也要读原文。
7. `표현의 쓰임을 확인하면서 읽다.` = 一边确认表达用法一边读。
