# 易混语法对比总表

本文件专门整理 TOPIK 阅读前几题和选项中容易混淆的语法。复习时不要只背中文，要看“功能”和“排除理由”。

---

## 1. 可能 / 不可能

| 语法 | 核心意思 | 语气 | 例句 |
|---|---|---|---|
| `-(으)ㄹ지도 모르다` | 也许……，可能…… | 推测，不确定 | 약속 시간에 늦을지도 몰라요. |
| `-(으)ㄹ 수도 있다` | 也可能……，有可能…… | 可能性 | 약속 시간에 늦을 수도 있어요. |
| `-(으)ㄹ 리가 없다` | 不可能…… | 强烈否定可能性 | 그렇게 쉬운 문제가 틀릴 리가 없어요. |

### 例句对比

```korean
지금 출발하지 않으면 약속 시간에 늦을지도 몰라요.
```
中文：现在不出发的话，可能会迟到。

```korean
지금 출발하지 않으면 약속 시간에 늦을 수도 있어요.
```
中文：现在不出发的话，也可能会迟到。

```korean
지금 출발하면 약속 시간에 늦을 리가 없어요.
```
中文：现在出发的话，不可能迟到。

## 2. 按照 / 如同 / 正如

| 语法 | 核心意思 | 常见中文 | 例句 |
|---|---|---|---|
| `-대로` | 按照……；正如…… | 按照、如……一样 | 예상한 대로 가격이 떨어졌다. |
| `-(으)ㄴ 것과 같이` | 和……一样 | 如……一样 | 예상한 것과 같이 가격이 떨어졌다. |
| `-처럼` | 像……一样 | 像……一样 | 새 옷처럼 깨끗해졌다. |

### 例句对比

```korean
전문가들이 예상한 대로 농산물 가격이 떨어지고 있다.
```
中文：正如专家预测的那样，农产品价格正在下降。

```korean
전문가들이 예상한 것과 같이 농산물 가격이 떨어지고 있다.
```
中文：和专家预测的一样，农产品价格正在下降。

```korean
더러워진 옷이 새 옷처럼 깨끗해졌다.
```
中文：变脏的衣服像新衣服一样干净了。

## 3. 时间相关：以来 / 时候 / 期间 / 之后

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-(으)ㄴ 지` | 做某事以来，过了多久 | 이사를 온 지 일 년이 됐다. |
| `-(으)ㄹ 때` | ……的时候 | 걸을 때 발이 편하다. |
| `-는 동안에` | 在……期间 | 수업을 듣는 동안에 졸았다. |
| `-(으)ㄴ 후` | ……之后 | 좌석을 선택한 후 결제하세요. |
| `-고 나서` | 做完A之后B | 설명을 듣고 나서 문제를 풀었다. |
| `-고 나서야` | 做完A之后才B | 설명을 듣고 나서야 이해했다. |

### 排除重点

- `온 지 일 년이 됐다` = 来了以后已经一年。
- `올 때` = 来的时候。
- `듣고 나서` = 听完之后。
- `듣고 나서야` = 听完之后才。

## 4. 过程中发生 / 做完之后

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-다가` | 做A过程中发生B | 휴대 전화를 보다가 역을 지나쳤다. |
| `-고서` | 做完A之后B | 숙제를 끝내고서 쉬었다. |
| `-고 나서` | 做完A之后B | 밥을 먹고 나서 공부했다. |

### 例句对比

```korean
휴대 전화를 보다가 내려야 할 역을 지나쳤다.
```
中文：看手机的时候，坐过站了。

```korean
휴대 전화를 보고서 집에 갔다.
```
中文：看完手机后回家了。

重点：`-다가` 是“过程中”，经常接意外结果；`-고서/-고 나서` 是“做完后”。

## 5. 逐渐变化 / 正在进行

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-아/어 가다` | 逐渐……下去 | 나뭇잎 색이 점점 붉게 변해 간다. |
| `-고 있다` | 正在…… | 농산물 가격이 떨어지고 있다. |

### 例句对比

```korean
색이 점점 붉게 변해 간다.
```
中文：颜色渐渐变红。

```korean
가격이 떨어지고 있다.
```
中文：价格正在下降。

重点：`-아/어 가다` 强调变化趋势；`-고 있다` 强调动作/状态正在进行。

## 6. 只要一……就 / 每次……都

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-기만 하면` | 只要一……就…… | 차를 타기만 하면 멀미를 한다. |
| `-(으)ㄹ 때마다` | 每次……的时候都 | 차를 탈 때마다 멀미를 한다. |

### 例句对比

```korean
동생은 차를 타기만 하면 멀미를 한다.
```
中文：弟弟只要一坐车就晕车。

```korean
동생은 차를 탈 때마다 멀미를 한다.
```
中文：弟弟每次坐车都晕车。

这两个常可互换，但 `-기만 하면` 更强调“一发生A，B就出现”。

## 7. 原因类：普通原因 / 负面原因 / 意外原因

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-기 때문에` | 因为…… | 시간이 부족하기 때문에 먼저 쉬운 문제부터 풀어야 한다. |
| `-는/은 탓에` | 因为……导致坏结果 | 잠을 못 잔 탓에 피곤하다. |
| `-는 바람에` | 因为意外A，导致B | 버스를 놓치는 바람에 늦었다. |
| `-느라고` | 因忙着A导致B不好 | 공부하느라고 늦게 잤다. |

### 排除重点

- `탓에` 带“都怪……”的负面感。
- `바람에` 常带意外、突发感。
- `느라고` 前面是自己做的动作，后面多为负面结果。

---

## 8. 目的 / 使令

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-고자` | 为了……，书面 | 실력을 확인하고자 모의고사를 봤다. |
| `-기 위해서` | 为了…… | 건강을 회복하기 위해서 충분히 잔다. |
| `-도록 하다` | 让……做 / 使……做 | 학생들이 알도록 설명했다. |

### 排除重点

```korean
한국 문화를 알게 되었다.
```
中文：变得了解韩国文化。

```korean
한국 문화를 알도록 했다.
```
中文：让别人了解韩国文化。

`-게 되다` 是“自己变得……”，`-도록 하다` 是“让别人/使某人……”。

---

## 9. 必须 / 可以 / 如果想

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-아/어야 하다` | 必须…… | 문법을 익혀야 한다. |
| `-아/어도 되다` | 可以…… | 조금 늦어도 된다. |
| `-(으)려면` | 如果想…… | 좋은 점수를 받으려면 실수를 줄여야 한다. |

### 例句对比

```korean
한국 문화를 알아야 한다.
```
中文：必须了解韩国文化。

```korean
한국 문화를 알아도 된다.
```
中文：可以了解韩国文化。

```korean
한국 문화를 알려면 한국 친구와 이야기해 보세요.
```
中文：如果想了解韩国文化，试着和韩国朋友聊聊。

## 10. 即使……也 / 就算做也没用

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-아/어도` | 即使……也…… | 어려워도 포기하지 마세요. |
| `-다고 해도` | 即使说…… / 即使……也 | 수리한다고 해도 오래 쓰기 어렵다. |
| `-아/어 봐야` | 即使试着做也……，多带“没用” | 수리해 봐야 오래 쓰기 어렵다. |

### 例句对比

```korean
이 컴퓨터는 수리한다고 해도 오래 쓰기 어렵다.
```
中文：这台电脑就算修了，也很难用很久。

```korean
이 컴퓨터는 수리해 봐야 오래 쓰기 어렵다.
```
中文：这台电脑就算试着修，也很难用很久。

`-아/어 봐야` 经常有“做了也没什么用”的感觉。

## 11. 判断 / 总结

| 语法 | 核心意思 | 例句 |
|---|---|---|
| `-(으)ㄴ 셈이다` | 算是……；等于…… | 거의 안 잔 셈이다. |
| `-거나 마찬가지이다` | 和……差不多 | 안 읽은 거나 마찬가지이다. |

### 例句对比

```korean
하루에 한 시간만 잤으니 거의 안 잔 셈이에요.
```
中文：一天只睡一个小时，所以几乎算是没睡。

```korean
책을 샀지만 읽지 않았으니 안 산 거나 마찬가지예요.
```
中文：虽然买了书但没读，所以和没买差不多。

## 13. 6月8日新增：完成后 / 之前 / 期间 / 一……就

| 语法 | 核心意思 | 例句 | 排除点 |
|---|---|---|---|
| `V-고 나서` | 做完A之后B | 밥을 먹고 나서 약을 먹었다. | A完成后才B |
| `V-기 전에` | 做A之前 | 밥을 먹기 전에 손을 씻었다. | B发生在A之前 |
| `V-는 동안` | 做A期间 | 밥을 먹는 동안 전화가 왔다. | A进行中发生B |
| `V-자마자` | 一A就B | 집에 도착하자마자 잤다. | A后马上B |
| `V-다가` | 做A中途转为B | 밥을 먹다가 전화를 받았다. | A未必完成 |

### 排除重点

- `먹고 나서`：吃完之后。
- `먹기 전에`：吃之前。
- `먹는 동안`：吃的期间。
- `먹자마자`：一吃完/一开始吃后马上。
- `먹다가`：吃着吃着。

---

## 14. 6月8日新增：提前做好 / 正在做 / 试着做 / 变得

| 语法 | 核心意思 | 例句 | 排除点 |
|---|---|---|---|
| `V-아/어 놓다` | 提前做好并保持状态 | 서류를 가방에 넣어 놓았다. | 做好并留着后面用 |
| `V-아/어 두다` | 事先做好 | 표를 미리 사 두었다. | 和 `놓다` 接近，偏预备 |
| `V-고 있다` | 正在做 | 지금 서류를 쓰고 있다. | 动作正在进行 |
| `V-아/어 보다` | 试着做 | 이 방법을 한번 써 보았다. | 体验、尝试 |
| `V-게 되다` | 变得、开始 | 그 일을 알게 되었다. | 状态变化或结果 |

### 排除重点

- `적어 놓다` = 写下来留着。
- `적고 있다` = 正在写。
- `적어 보다` = 试着写。
- `알게 되다` = 变得知道、开始知道。

## 15. 6月8日新增：不得不 / 可能 / 不可能 / 可以

| 语法 | 核心意思 | 例句 | 排除点 |
|---|---|---|---|
| `V-아/어야만 하다` | 必须得…… | 오늘 안에 제출해야만 한다. | 强义务 |
| `V-(으)ㄹ 수밖에 없다` | 只能……；不得不…… | 약속을 미룰 수밖에 없었다. | 没有别的选择 |
| `V-(으)ㄹ 수도 있다` | 也可能…… | 늦을 수도 있다. | 可能性 |
| `V-(으)ㄹ 리가 없다` | 不可能…… | 그가 모를 리가 없다. | 强烈否定 |
| `V-아/어도 되다` | 可以…… | 조금 늦어도 된다. | 允许 |

### 排除重点

- `미뤄야만 했다` ≈ `미룰 수밖에 없었다`：必须推迟 / 不得不推迟。
- `미룰 수도 있었다`：也可能推迟，不是“不得不”。
- `미룰 리가 없었다`：不可能推迟，方向相反。
- `미뤄도 되었다`：可以推迟，语气弱。

---

## 16. 6月8日新增：目的 / 原因 / 之前 / 条件

| 语法 | 核心意思 | 例句 | 排除点 |
|---|---|---|---|
| `V-기 위해` | 为了…… | 건강을 지키기 위해 운동한다. | 目的 |
| `V-기 때문에` | 因为…… | 비가 오기 때문에 집에 있었다. | 原因 |
| `V-기 전에` | 在……之前 | 자기 전에 책을 읽었다. | 时间 |
| `V-기만 하면` | 只要……就…… | 만나기만 하면 싸운다. | 条件/反复结果 |
| `V-는 대신에` | 代替…… | 밥을 먹는 대신에 빵을 먹었다. | 替代 |

### 排除重点

- `막기 위해` = 为了防止。
- `막기 때문에` = 因为防止，不自然或语义不对。
- `막기 전에` = 防止之前。
- `막기만 하면` = 只要防止就。

## 17. 6月8日新增：对比 / 让步 / 普通转折 / 反而

| 连接词 | 核心意思 | 例句 | 排除点 |
|---|---|---|---|
| `반면` | 另一方面；相反 | 가격은 비싸다. 반면 품질은 좋다. | A/B对比 |
| `그래도` | 即使如此、不过还是 | 비가 왔다. 그래도 갔다. | 困难后仍继续 |
| `하지만` | 但是 | 비싸다. 하지만 좋다. | 普通转折 |
| `오히려` | 反而 | 쉬울 줄 알았는데 오히려 어려웠다. | 和预想相反 |
| `따라서` | 因此 | 위험하다. 따라서 조심해야 한다. | 正式结论 |

### 排除重点

- `반면`：比较两个方面。
- `그래도`：前面不好，后面仍然做。
- `하지만`：普通“但是”。
- `오히려`：结果和预想相反。
- `따라서`：总结前文，提出结论。

---

## 18. 6月8日新增：好像 / 按照 / 如同 / 原样照做

| 语法 | 核心意思 | 例句 | 排除点 |
|---|---|---|---|
| `마치 ... 것처럼` | 好像……一样 | 마치 현장에 있는 것처럼 느껴졌다. | 比喻、感觉 |
| `-처럼` | 像……一样 | 새 옷처럼 깨끗하다. | 简单比喻 |
| `-대로` | 按照…… | 설명서대로 따라 했다. | 按规则或方式 |
| `-(으)ㄴ 것과 같이` | 和……一样 | 예상한 것과 같이 비가 왔다. | 事实相同 |
| `그대로 따르다` | 照原样跟随、照着做 | 설명을 그대로 따랐다. | 不改变，照着做 |

### 排除重点

- `마치 직접 보는 것처럼 느껴졌다` = 感觉好像亲眼看见一样。
- `설명대로 했다` = 按照说明做。
- `설명을 그대로 따랐다` = 原样照着说明做。
- `예상한 것과 같이` = 和预想一样。

---

## 19. 6月8日新增：同时 / 完成后 / 期间 / 随着

| 语法 | 核心意思 | 例句 | 排除点 |
|---|---|---|---|
| `V-면서` | 一边……一边……；同时 | 음악을 들으면서 걸었다. | 两动作同时或伴随 |
| `V-고 나서` | 做完之后 | 음악을 듣고 나서 잤다. | A完成后B |
| `V-는 동안` | 在……期间 | 음악을 듣는 동안 쉬었다. | A期间B |
| `A-아/어지면서` | 随着变得…… | 날씨가 추워지면서 사람이 줄었다. | 状态变化伴随结果 |
| `V-다가` | 做着做着 | 음악을 듣다가 잠이 들었다. | 中途转变 |

### 排除重点

- `들으면서 적다` = 一边听一边写。
- `듣고 나서 적다` = 听完后写。
- `듣는 동안 적다` = 听的时候写，偏期间。
- `추워지면서` = 随着变冷。

## 当前最该优先背的易混组

1. `-(으)ㄹ지도 모르다 / -(으)ㄹ 수도 있다 / -(으)ㄹ 리가 없다`
2. `-대로 / -(으)ㄴ 것과 같이 / -처럼 / 마치 ... 것처럼 / 그대로 따르다`
3. `-다가 / -고서 / -고 나서 / -기 전에 / -는 동안 / -자마자`
4. `-아/어 가다 / -고 있다 / -아/어 놓다 / -아/어 두다 / -아/어 보다 / -게 되다`
5. `-기만 하면 / -(으)ㄹ 때마다`
6. `-는/은 탓에 / -는 바람에 / -느라고 / -기 때문에`
7. `-기 위해 / -기 때문에 / -기 전에 / -기만 하면 / -는 대신에`
8. `그래도 / 그러면 / 따라서 / 반면 / 하지만 / 오히려`
9. `-아/어야만 하다 / -(으)ㄹ 수밖에 없다 / -(으)ㄹ 수도 있다 / -(으)ㄹ 리가 없다 / -아/어도 되다`
10. `-다고 해도 / -아/어 봐야 / -아/어도`
11. `-(으)ㄴ 셈이다 / -거나 마찬가지이다`
