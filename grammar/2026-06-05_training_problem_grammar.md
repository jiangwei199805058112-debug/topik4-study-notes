# 2026-06-05 训练暴露语法整理

来源：6月5日后半段句子训练。

重点记录反复训练后仍然不稳定的语法、变化形和固定句型。

---

## 1. `V-아/어야 해요`

### 核心意思

必须……；应该……；得……。

### 本轮暴露问题

用户对 `가야 해요`, `봐야 해요`, `골라야 해요`, `구별해야 해요`, `확인해야 해요` 仍需加强。

### 例句

```korean
세탁소에 맡긴 옷을 찾으러 가야 해요.
```

中文：得去取交给洗衣店的衣服。

```korean
선택지만 보지 말고 지문도 다시 봐야 해요.
```

中文：不要只看选项，原文也要重新看。

```korean
맞는 답을 골라야 해요.
```

中文：必须选择正确答案。

```korean
질문과 지문을 구별해야 해요.
```

中文：必须区分问题和阅读原文。

```korean
영수증을 확인해야 해요.
```

中文：必须确认收据。

### 易错点

`-아/어야 해요` 是义务和必要，不是“可以”。看到 `-야 해요` 先反应成“必须、得”。

## 2. `V-(으)려고 하다`

### 核心意思

想要……；打算……；正准备……。

### 本轮暴露问题

`고르려고 하다`, `찍으려고 하다`, `포기하려고 하다`, `피하려고 하다` 都需要马上反应为“想做某事”。

### 变化规则

| 动词 | 变化 | 中文 |
|---|---|---|
| 고르다 | 고르려고 하다 | 想选择 |
| 찍다 | 찍으려고 하다 | 想蒙答案 |
| 포기하다 | 포기하려고 하다 | 想放弃 |
| 피하다 | 피하려고 하다 | 想避开 |

### 例句

```korean
선택지만 보고 답을 고르려고 하면 실수할 수 있어요.
```

中文：如果想只看选项就选答案，可能会出错。

```korean
어려운 문제가 나오면 바로 찍으려고 하지 마세요.
```

中文：遇到难题时，不要马上想蒙答案。

```korean
어려운 문장을 바로 포기하려고 하면 글의 흐름을 파악하기 어려워요.
```

中文：如果一看到难句就想放弃，就很难把握文章脉络。

### 易错点

`-려고 하다` 表示意图，不表示已经完成。`찍으려고 하다` 是“想蒙答案”，不是“已经蒙了答案”。

## 3. `V-(으)ㄹ 수 있다`

### 核心意思

能……；可以……；有能力做某事。

### 本轮暴露问题

`풀 수 있다` 要反应成“能做出来、能解题”，不要只理解成抽象的“能够解决”。

### 例句

```korean
글의 흐름을 파악하면 어려운 문제도 풀 수 있어요.
```

中文：如果把握文章脉络，难题也能做出来。

```korean
질문의 뜻을 이해하면 맞는 답을 고를 수 있어요.
```

中文：如果理解问题的意思，就能选出正确答案。

### 易错点

`문제를 풀 수 있다` 在 TOPIK 阅读语境里优先理解成“题能做出来”，不是单纯“能解决问题”。

---

## 4. `V-(으)러 가다`

### 核心意思

为了做某事而去；去做……。

### 本轮暴露问题

`찾으러 가다` 要理解成“去取、去拿回来”，不是“去寻找一件不知道在哪里的东西”。

### 例句

```korean
세탁소에 맡긴 옷을 찾으러 가야 해요.
```

中文：得去取交给洗衣店的衣服。

```korean
필요한 자료를 찾으러 도서관에 갔어요.
```

中文：去图书馆找需要的资料。

### 易错点

`V-(으)러 가다` 的重点是“去”的目的。`찾으러 가다` 在洗衣店、快递、保管物语境里常是“去取”。

---

## 5. `어렵다 → 어려운 N`

### 核心意思

`어렵다` = 难。

`어려운 N` = 难的N。

### 本轮暴露问题

`어려운` 被反复误看成“男的”。必须固定反应为“难的”。

### 例句

```korean
어려운 문장을 계속 피하면 실수할 수밖에 없어요.
```

中文：如果一直避开难句，就只能出错。

```korean
어려운 문제도 글의 흐름을 보면 풀 수 있어요.
```

中文：难题也可以通过看文章脉络做出来。

### 易错点

看到 `어려운 문장`, `어려운 문제`, `어려운 글`, `어려운 표현`，第一反应都要是“难的……”。

---

## 6. `N만` / `N도`

### 核心意思

`만` = 只、仅仅。

`도` = 也。

### 本轮暴露问题

`선택지만 보고` 和 `지문도 다시 읽다` 是阅读训练高频动作。

### 例句

```korean
선택지만 보고 답을 고르면 안 돼요.
```

中文：不能只看选项就选答案。

```korean
질문만 보지 말고 지문도 다시 읽어야 해요.
```

中文：不要只看问题，原文也要重新读。

### 易错点

`만` 会缩小范围，`도` 会补充范围。阅读题里经常用来提醒不要只看选项，还要回到原文。

---

## 7. `V-면서`

### 核心意思

一边……一边……；两个动作同时进行。

### 本轮暴露问题

`확인하면서 읽다` 要反应为“一边确认一边读”。

### 例句

```korean
표현의 쓰임을 확인하면서 읽어야 해요.
```

中文：要一边确认表达的用法，一边阅读。

```korean
질문의 뜻을 생각하면서 지문을 다시 읽어 보세요.
```

中文：请一边思考问题的意思，一边重新读原文。

### 易错点

`-면서` 不只是机械翻译成“一边”，更重要的是判断前后两个动作是否同时进行。

---

## 8. `V-지 말고`

### 核心意思

不要A，而要B。

### 本轮暴露问题

阅读训练里需要固定反应：不要马上蒙答案、不要马上放弃、不要只看选项。

### 例句

```korean
바로 답을 찍지 말고 지문도 다시 읽어야 해요.
```

中文：不要马上蒙答案，原文也要重新读。

```korean
어려운 문장을 피하지 말고 반복해서 읽어 보세요.
```

中文：不要避开难句，反复读试看。

### 易错点

`-지 말고` 后面通常接真正应该做的动作。句子重点常在后半句。

## 今日优先反应顺序

| 优先级 | 语法/变化形 | 必须反应出来的中文 |
|---|---|---|
| A | `V-아/어야 해요` | 必须、应该、得 |
| A | `V-(으)려고 하다` | 想要、打算 |
| A | `V-(으)ㄹ 수 있다` | 能、可以 |
| A | `어렵다 → 어려운 N` | 难的N |
| B | `V-(으)러 가다` | 为了做某事而去 |
| B | `N만` | 只、仅仅 |
| B | `N도` | 也 |
| B | `V-면서` | 一边……一边…… |
| B | `V-지 말고` | 不要A，而要B |

---

## 最低背诵句

1. `맞는 답을 골라야 해요.` = 必须选择正确答案。
2. `어려운 문장을 피하지 마세요.` = 不要避开难句。
3. `선택지만 보고 답을 찍지 마세요.` = 不要只看选项就蒙答案。
4. `글의 흐름을 파악하면 문제를 풀 수 있어요.` = 把握文章脉络就能做题。
5. `표현의 쓰임을 확인하면서 읽어야 해요.` = 要一边确认表达用法一边读。
6. `질문과 지문을 구별해야 해요.` = 必须区分问题和原文。
