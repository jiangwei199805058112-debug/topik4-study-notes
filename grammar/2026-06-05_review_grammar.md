# 2026-06-05 回炉语法表

来源：6月5日 TOPIK 风格短文精读、题目审核，以及 `-면서 / 속에서 / 전혀 기억이 나지 않다 / 단순히 A가 아니라 B` 等问题复盘。

目标：不是只背中文翻译，而是看到句子后能马上判断语法功能和句子关系。

---

## 1. `-면서`

### 核心意思

1. 一边A，一边B。
2. 随着A，B也发生变化。

### 本轮句子

```korean
중심 내용을 파악하면서 모르는 표현을 함께 정리해야 한다.
```

中文：要一边把握中心内容，一边一起整理不懂的表达。

### 拆解

```text
파악하다 → 파악하면서
```

这里的 `-면서` 表示两个动作同时进行：`중심 내용을 파악하다` 和 `모르는 표현을 정리하다`。

### 例句

```korean
예문을 읽으면서 단어의 쓰임을 익혀야 해요.
```

中文：要一边读例句，一边掌握单词的用法。

```korean
한국어 글을 많이 읽으면서 표현이 자연스러워졌어요.
```

中文：随着读了很多韩语文章，表达变得自然了。

### 易错点

`-면서` 不一定只翻译成“一边……一边……”。如果前后是变化关系，可以理解成“随着……”。

## 2. `N 속에서`

### 核心意思

在N之中；在某个范围、环境、语境里面。

### 本轮句子

```korean
문장 속에서 표현을 익히는 것이 더 효과적이다.
```

中文：在句子中掌握表达更有效。

### 拆解

```text
문장 + 속 + 에서 → 문장 속에서
```

`속` 不只表示物理空间里的“里面”，也可以表示抽象范围。

### 例句

```korean
생활 속 표현도 함께 익혀야 해요.
```

中文：生活中的表达也要一起掌握。

```korean
글 속에서 중심 내용을 파악해야 해요.
```

中文：要在文章中把握中心内容。

### 易错点

不要把 `속` 只理解成“盒子里面、房间里面”这种物理空间。TOPIK 文章里常见 `문장 속`, `글 속`, `생활 속` 这类抽象范围。

## 3. `전혀 ... 않다`

### 核心意思

完全不……；一点也不……。

### 本轮句子

```korean
설명이 전혀 기억이 나지 않아요.
```

中文：说明完全想不起来。

### 拆解

```text
전혀 + 기억이 나지 않다
```

`전혀` 后面通常接否定表达，比如 `모르다`, `없다`, `않다`, `기억이 나지 않다`。

### 例句

```korean
처음에는 이 표현을 전혀 몰랐어요.
```

中文：一开始完全不知道这个表达。

```korean
어제 배운 문법이 전혀 기억이 나지 않아요.
```

中文：昨天学的语法完全想不起来。

### 易错点

`기억이 나지 않다` 是“想不起来”，不是“没有背下来”。`전혀 기억이 나지 않다` 的反应要定成“完全想不起来”。

## 4. `단순히 A가 아니라 B`

### 核心意思

不只是A，而是B；重点不在A，而在B。

### 本轮句子

```korean
단순히 단어를 외우는 것이 아니라 표현을 익혀야 한다.
```

中文：不只是背单词，而要掌握表达。

### 拆解

```text
단순히 A가 아니라 B
```

`단순히` 表示“仅仅、只是”。`A가 아니라 B` 表示否定A、强调B。

### 例句

```korean
한국어 공부는 단순히 문법을 외우는 것이 아니라 문장 속에서 쓰임을 익히는 것이다.
```

中文：韩语学习不只是背语法，而是在句子中掌握用法。

```korean
이 문제는 단순히 단어 뜻을 묻는 것이 아니라 중심 내용을 파악하는 문제예요.
```

中文：这道题不只是问单词意思，而是把握中心内容的题。

### 易错点

不要理解成“单纯地不做A做B”。它的句子重心是：A还不够，真正要强调的是B。

## 5. `V-(으)ㄹ 수밖에 없다`

### 核心意思

只能……；不得不……；除了这样没有别的办法。

### 本轮句子

```korean
어려운 문장을 계속 피하면 실수할 수밖에 없다.
```

中文：如果一直避开难句，就只能出错。

### 拆解

```text
실수하다 → 실수할 수밖에 없다
```

`수` 是“方法、可能性”，`밖에 없다` 是“除此之外没有”。合在一起就是“只能……”。

### 例句

```korean
복습하지 않으면 같은 표현에서 또 틀릴 수밖에 없어요.
```

中文：如果不复习，就只能在同一个表达上再次出错。

```korean
문장을 많이 보지 않으면 표현이 어색하게 느껴질 수밖에 없어요.
```

中文：如果不多看句子，表达只能感觉别扭。

### 易错点

`수밖에 없다` 不是“有机会”，而是“没有别的选择，只能这样”。

## 6. `V-아/어지다`：`느껴지다`, `자연스러워지다`

### 核心意思

变得……；被感觉为……。

### 本轮句子

```korean
많이 공부한 것처럼 느껴지지만 실제로는 기억이 나지 않을 때가 있다.
```

中文：有时感觉像学了很多，但实际上想不起来。

```korean
어느 순간 발음이 자연스러워질 수 있다.
```

中文：某个时候发音可能会自然起来。

### 拆解

```text
느끼다 → 느껴지다
자연스럽다 → 자연스러워지다
```

`느껴지다` 更偏“感觉起来、被感觉到”。`자연스러워지다` 表示状态变化：从不自然变得自然。

### 例句

```korean
예문과 함께 보면 표현이 더 쉽게 느껴져요.
```

中文：和例句一起看，表达会感觉更容易。

```korean
계속 따라 하면 말하기가 자연스러워져요.
```

中文：如果持续跟读，说话会自然起来。

### 易错点

`느껴지다` 不要按主动的“我感觉”死翻，要反应成“感觉起来、被感觉为”。

## 今日优先复习顺序

| 优先级 | 语法/表达 | 必须反应出来的中文 |
|---|---|---|
| A | `-면서` | 一边……一边…… / 随着…… |
| A | `N 속에서` | 在N之中、在N范围里 |
| A | `전혀 ... 않다` | 完全不…… |
| A | `단순히 A가 아니라 B` | 不只是A，而是B |
| A | `V-(으)ㄹ 수밖에 없다` | 只能、不得不 |
| B | `V-아/어지다` | 变得…… / 被感觉为…… |
| B | `V-(으)러 가다` | 为了做某事而去 |
| B | `따라서` | 因此、所以 |

## 反应训练

看到下面表达时，先反应语法功能，再翻译整句。

1. `문장 속에서` = 在句子中。
2. `파악하면서` = 一边把握，一边……。
3. `전혀 기억이 나지 않다` = 完全想不起来。
4. `단순히 단어를 외우는 것이 아니라` = 不只是背单词，而是……。
5. `실수할 수밖에 없다` = 只能出错、不得不出错。
6. `자연스러워지다` = 变得自然。
