# 题目去重与变式生成规则

目的：避免复习时反复出现完全相同的题目，导致“记住答案”而不是“掌握知识点”。

---

## 1. 总原则

复习可以重复同一个知识点，但不能重复同一道题。

```text
可以重复：그래서 / 그래도 / 그러면 / 따라서
不能重复：完全相同的句子、相同选项、相同答案位置
```

正确做法：

```text
同知识点 → 换场景 → 换句子 → 换选项顺序 → 换干扰项
```

---

## 2. 题目记录规则

每一道练习题都应记录一个题目 ID。

题目 ID 推荐格式：

```text
YYYY-MM-DD-题型-序号-目标点
```

示例：

```text
2026-06-07-CONN-001-그래도
2026-06-07-GRAM-003-대로
2026-06-07-AD-002-세탁소
2026-06-07-READ-001-중심생각
```

---

## 3. 每题需要记录的信息

| 字段 | 说明 |
|---|---|
| ID | 题目编号 |
| 日期 | 第一次出现日期 |
| 题型 | 连接词、语法、广告、阅读、顺序排列等 |
| 目标点 | 要测试的词/语法/题型 |
| 题干摘要 | 韩文题干简写 |
| 正确答案 | 正确选项或答案 |
| 干扰项 | 错误选项类型 |
| 是否可复用 | 原题是否可以再次出现 |
| 下次规则 | 原句复现、变式、综合检测 |

---

## 4. 去重标准

### 禁止重复

以下情况视为重复，应避免再次出现：

1. 题干完全相同。
2. 四个选项完全相同。
3. 正确答案位置完全相同且句子几乎相同。
4. 广告题关键词组合完全相同。
5. 阅读短文主题、句子和选项都基本相同。

### 允许重复

以下情况允许：

1. 同一个语法点重复测试。
2. 同一个连接词重复测试。
3. 同一种题型重复出现。
4. 同一个 TOPIK 高频场景重复出现，但必须换句子。

---

## 5. 复习变式等级

| 等级 | 用途 | 是否允许原句 |
|---|---|---|
| D0 原句确认 | 刚学完，确认理解 | 可以少量原句 |
| D1 变式复习 | 第二天复习 | 不允许原句 |
| D3 混合题 | 第三天复习 | 不允许原句 |
| D7 综合题 | 一周后检测 | 不允许原句 |
| D14 长文检测 | 两周后检测 | 不允许原句 |

原则：D1 以后不再重复原句。

---

## 6. 连接词题变式规则

以 `그래도` 为例。

### 不要重复

```korean
비가 많이 왔다. 그래도 행사는 취소되지 않았다.
```

### 可以变式

```korean
문제가 어려웠다. 그래도 바로 답을 찍지 않았다.
```

```korean
시간이 부족했다. 그래도 지문을 끝까지 읽었다.
```

```korean
모르는 단어가 많았다. 그래도 중심 내용을 파악하려고 했다.
```

目标不变：`그래도 = 即使如此、不过还是`。  
句子必须变。

---

## 7. 广告/说明题变式规则

同一个场景可以重复，但关键词组合要变化。

### 鞋/구두 场景

旧题关键词：

```text
발이 편하다 + 가볍다 + 디자인
```

变式关键词：

```text
오래 걸어도 편하다 + 발이 아프지 않다 + 출근할 때 신기 좋다
```

### 洗衣店/세탁소 场景

旧题关键词：

```text
더러워진 옷 + 두꺼운 이불 + 맡기다
```

变式关键词：

```text
얼룩 제거 + 양복 세탁 + 깨끗하게 다림질
```

---

## 8. 阅读题变式规则

阅读题可以重复题型，但短文内容必须变。

### 可以重复题型

```text
중심 생각
내용 일치
빈칸 연결어
밑줄 의미
```

### 必须变化

1. 短文主题要换。
2. 关键词要换。
3. 选项顺序要换。
4. 正确答案位置要换。
5. 干扰项要重新设计。

---

## 9. 出题前检查流程

每次出题前先检查：

1. `daily/` 中最近 3 天是否出现过相同句子。
2. `practice/used_question_registry.md` 中是否记录过相同题干。
3. 如果目标点相同，必须换场景或换句子。
4. 如果用户指出重复，要把该题标记为“禁止复用”。

---

## 10. 用户反馈后的处理

用户反馈“这题重复”时：

1. 不再解释该题。
2. 将该题记录到 `used_question_registry.md`。
3. 以后同题干禁止再次使用。
4. 只保留知识点，重新生成新场景题。

---

## 11. 题目生成最低要求

以后生成复习题必须满足：

1. 题型可以重复，内容不能完全重复。
2. 选项要有干扰性，不能一眼看出答案。
3. 同一个知识点至少准备 3 个不同语境。
4. 正确答案位置要轮换。
5. 每次练习结束后，错题和重复题都要记录到每日文件。

---

## 12. 难度与干扰项规则

用户指出：如果四个选项差距过大，就算不读题也能猜出答案，不能有效训练。

以后新内容即时练习和复习卷都按以下规则提高难度：

1. 不能只把今天刚学的语法放进选项。
2. 选项必须混入之前学过的相近语法和相近表达。
3. 四个选项都要与题干语境有关，不能出现明显无关的选项。
4. 每题可以加入 1 到 2 个可根据上下文推断的新词，但不能让生词成为唯一解题障碍。
5. 选项应考察语义功能，而不是只看形式就能秒选。
6. 主题要轮换，不能总是 TOPIK、发音、复习、学习方法。

---

## 13. 选项干扰项设计模板

### 语法填空题

四个选项应尽量来自相近功能：

| 目标 | 可用干扰项 |
|---|---|
| `V-고 나서` | `V-기 전에`, `V-는 동안`, `V-자마자`, `V-다가` |
| `V-아/어 놓다` | `V-아/어 두다`, `V-고 있다`, `V-아/어 보다`, `V-게 되다` |
| `-는 바람에` | `-는 탓에`, `-기 때문에`, `-는 김에`, `-기 위해` |
| `-ㄹ 수밖에 없다` | `-아/어야만 하다`, `-ㄹ 리가 없다`, `-ㄹ 수도 있다`, `-아/어도 되다` |
| `-기 위해` | `-기 때문에`, `-기 전에`, `-기만 하면`, `-는 대신에` |
| `반면` | `하지만`, `그래도`, `오히려`, `또한` |

### 广告题

四个选项都必须属于可能场景，不能明显无关。

例如不能这样：

```text
이불 / 은행 / 자전거 / 컴퓨터
```

应改成：

```text
이불 / 침대 / 잠옷 / 세탁소
```

### 阅读题

错误选项应有明确类型：

1. 原文相反。
2. 原文未提及。
3. 偷换主体。
4. 范围扩大或缩小。
5. 因果关系颠倒。

---

## 14. 新内容练习分两档

新语法刚讲完后，不直接只做简单识别题。

### A档：确认题

目的：确认基础意思。数量少，最多 2 到 3 题。

### B档：TOPIK型混合题

目的：真正检测能否区分相近语法。必须混入旧语法和干扰项。

以后用户如果说“太简单”，立即切换到 B档。
