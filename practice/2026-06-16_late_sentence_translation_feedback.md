## 2. 第一轮重点句修正

| 韩语 | 修正中文 | 检测点 |
|---|---|---|
| 대출 금리가 떨어지면서 부동산 시장이 살아나기 시작했다. | 随着贷款利率下降，房地产市场开始复苏/回暖。 | 대출 금리, 살아나다 |
| 대출 금리 하락세에도 불구하고 거래는 크게 늘지 않았다. | 尽管贷款利率呈下降趋势，交易并没有大幅增加。 | -에도 불구하고 |
| 모처럼 찾아온 황금연휴에 여행 상품을 찾는 사람이 많아졌다. | 难得到来的黄金连休期间，寻找旅游产品的人变多了。 | 찾아오다 vs 찾다 |
| 갑자기 쏟아진 비로 차량이 엉금엉금 움직이면서 출근길이 막혔다. | 因为突然下起的大雨，车辆慢吞吞地移动，上班路堵住了。 | 갑자기 + 쏟아진, 엉금엉금 |
| 태풍 피해 복구가 지지부진하자 주민들이 답답해하고 있다. | 台风灾害恢复进展缓慢，居民们感到郁闷/憋闷。 | 복구, 지지부진하다, 답답해하다 |
| 복구가 잘 진행되지 않아 속 터지는 주민들이 대책을 요구했다. | 因为恢复进展不顺利，急得要命的居民们要求对策。 | 속 터지다, 요구하다 |
| 관중을 모으려고 공연 전부터 적극적으로 홍보했다. | 为了吸引观众，从演出前就开始积极宣传。 | 모으려고 |
| 콘서트를 열자 수많은 관중이 몰렸다. | 一举行演唱会，大批观众就涌来了。 | -자, 몰리다 |
| 규제가 부동산 시장에 영향을 미쳤다는 지적이 나왔다. | 有指出称，管制对房地产市场产生了影响。 | 영향을 미치다, 지적이 나오다 |
| 시장을 활성화하려고 금리 안정화 대책이 논의되고 있다. | 为了激活市场，利率稳定化对策正在被讨论。 | 활성화하려고, 논의되다 |

## 3. 第二轮 5句小测试批改

### 原句 1

```korean
대출 금리가 떨어지면서 시장이 살아나기 시작했다.
```

用户译文：随着贷款利率的下降，市场开始恢复。

判定：正确。

更自然中文：随着贷款利率下降，市场开始复苏/回暖。

---

### 原句 2

```korean
팬들이 한꺼번에 몰리면서 공연장이 혼잡해졌다.
```

用户译文：粉丝的一下子跟随者粉丝一下子涌了过来，表演场变得混乱。

判定：部分正确。

修正中文：粉丝们一下子涌来，演出场馆变得拥挤/混乱了。

重点：

```text
한꺼번에 몰리다 = 一下子聚集、一下子涌来
혼잡해지다 = 变得拥挤、混乱
```

---

### 原句 3

```korean
갑자기 쏟아진 비 때문에 출근길이 막혔다.
```

用户译文：突然倾盆大雨的原因，上班路上很堵。

判定：基本正确。

修正中文：因为突然下起的大雨，上班路堵了。

注意：

```text
쏟아진 비 = 倾盆而下的雨 / 下得很大的雨
갑자기 쏟아진 비 = 突然下起的大雨
```

---

### 原句 4

```korean
태풍 피해 복구가 잘 진행되지 않고 있다.
```

用户译文：台风灾后恢复的不是进行得很好。

判定：意思接近，中文顺序不自然。

修正中文：台风灾害恢复进展不顺利。

重点：

```text
복구가 잘 진행되지 않다 = 恢复进展不顺利
```

---

### 原句 5

```korean
원작 작가에 대한 관심이 높아졌다.
```

用户译文：对原作者的关心增多。

判定：基本正确。

修正中文：对原作作者的关注度提高了。

## 4. 第三轮回炉测试题

只测三个不稳点：

```text
한꺼번에 몰리다 = 一下子聚集、涌来
혼잡해지다 = 变得拥挤、混乱
복구가 잘 진행되지 않다 = 恢复进展不顺利
```

### 韩语 → 中文

```text
1. 팬들이 한꺼번에 몰려서 공연장이 혼잡해졌다.
2. 대중교통 이용자가 한꺼번에 몰리면서 역 주변이 혼잡해졌다.
3. 태풍 피해 복구가 지지부진하다.
4. 복구가 잘 진행되지 않아 주민들이 답답해하고 있다.
5. 갑자기 쏟아진 비 때문에 도로가 혼잡해졌다.
```
