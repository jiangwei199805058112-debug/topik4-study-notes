## 1. 练习目标

本轮专门回炉以下词和语法：

```text
솔직하게 = 坦率地、诚实地
그럼에도 불구하고 = 尽管如此
-다는 지적이 나오다 = 有人指出…… / 出现……的批评意见
한편 = 另一方面
금세 = 很快、马上
출근하던 사람들 = 当时正在上班的人们
-에도 불구하고 = 尽管……仍然……
커지고 있다 = 正在变大、正在增加
```

| 빠른 복구와 실질적인 대책을 요구하다 | 要求快速恢复和实际性的对策 |

## 4. 易混对比

| 韩语 | 中文 | 对比 |
|---|---|---|
| 시민 | 市民 | 城市、社会公共语境 |
| 주민 | 居民 | 某地区、某小区、受灾地区的人 |
| 신뢰를 얻다 | 获得信任 | 不是“失去信任” |
| 신뢰를 잃다 | 失去信任 | 与 얻다 相反 |
| 솔직하게 | 坦率地、诚实地 | 隐藏 = 숨기다 |
| 요구하다 | 要求 | 不是“有需求” |
