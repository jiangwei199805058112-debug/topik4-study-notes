# 已用题目登记表

目的：记录已经做过的题目，避免后续复习时原题重复出现。

---

## 使用规则

1. 每次生成练习题后，如果该题用于正式复习，应登记到这里。
2. D0 当天原句可以少量复现；D1 之后不再复用原句。
3. 如果用户指出某题重复，必须标记为“禁止复用”。
4. 后续可以继续考同一知识点，但必须换题干、换场景、换选项顺序。

---

## 字段说明

| 字段 | 说明 |
|---|---|
| ID | 题目编号 |
| 日期 | 第一次使用日期 |
| 题型 | CONN/GRAM/AD/READ/ORDER 等 |
| 目标点 | 被考察的词、语法或题型 |
| 题干摘要 | 韩文题干或关键词摘要 |
| 正确答案 | 正确选项 |
| 状态 | 可变式 / 禁止原题复用 / 已指出重复 |
| 备注 | 错因或后续规则 |
