## 2. 发现并修正的问题

### 2.1 `쏟아진 비` 不能单独翻成“突然下起的大雨”

| 原整理 | 问题 | 修正后 |
|---|---|---|
| `쏟아진 비 = 突然下起的大雨` | `쏟아진` 本身没有“突然”的意思 | `쏟아진 비 = 倾盆而下的雨 / 下得很大的雨` |
| `갑자기 쏟아진 비 = 突然下起的大雨` | 正确 | 保留 |
| `기습 폭우 = 갑자기 많이 쏟아진 비` | 正确 | 保留 |

记忆规则：

```text
쏟아지다 = 倾泻、下得很大
쏟아진 비 = 倾盆而下的雨 / 下得很大的雨
갑자기 = 突然
갑자기 쏟아진 비 = 突然下起的大雨
기습 폭우 = 突然暴雨
```

## 3. 需要保留但补充语境的表达

### 3.1 `모으다`

| 韩语 | 基本意思 | 新闻题语境 |
|---|---|---|
| 모으다 | 聚集、召集、收集 | 吸引、招来 |
| 관중을 모으다 | 聚集观众 | 吸引观众 |
| 관중이 몰리다 | 观众聚集 | 观众涌来 |

记忆规则：

```text
모으다 = 我把人聚过来、吸引过来
몰리다 = 人自己涌过来
```
