# TOPIK 91届 阅读 1–4题错题整理

这组题主要暴露的问题：不是单词完全不会，而是固定语法、固定搭配和排除项语法不熟。

---

## 第1题

### 原题

```korean
나는 오래전에 설악산을 (      ).
```

选项：

1. 등산하고 싶다
2. 등산해도 된다
3. 등산할 것 같다
4. 등산한 적이 있다

### 正确答案

```korean
등산한 적이 있다
```

### 正确句子

```korean
나는 오래전에 설악산을 등산한 적이 있다.
```

中文：我很久以前爬过雪岳山。

### 判断关键

`오래전에` = 很久以前，表示过去时间，所以后面要接过去经历。

### 选项语法整理

| 选项 | 语法 | 核心意思 | 本题意思 | 是否正确 |
|---|---|---|---|---|
| 등산하고 싶다 | `-고 싶다` | 想做…… | 想爬山 | 否 |
| 등산해도 된다 | `-아/어도 되다` | 可以…… | 可以爬山 | 否 |
| 등산할 것 같다 | `-(으)ㄹ 것 같다` | 好像会…… | 好像会爬山 | 否 |
| 등산한 적이 있다 | `-(으)ㄴ 적이 있다` | 有过……经历 | 爬过山 | 是 |

## 第2题

### 原题

```korean
새집으로 (      ) 가구를 새로 샀다.
```

选项：

1. 이사한 지
2. 이사하거든
3. 이사하려면
4. 이사하고 나서

### 正确答案

```korean
이사하고 나서
```

### 正确句子

```korean
새집으로 이사하고 나서 가구를 새로 샀다.
```

中文：搬到新房以后，重新买了家具。

### 判断关键

`샀다` 是过去式，表示已经买了。句子逻辑是：搬家之后 → 买家具。

### 选项语法整理

| 选项 | 语法 | 核心意思 | 常见搭配/后续 | 是否正确 |
|---|---|---|---|---|
| 이사한 지 | `-(으)ㄴ 지` | 做某事以来 | `이사한 지 3개월 됐다` | 否 |
| 이사하거든 | `-거든` | 如果……的话 | 后面常接建议/命令 | 否 |
| 이사하려면 | `-(으)려면` | 如果想…… | 后面常接 `해야 하다/필요하다` | 否 |
| 이사하고 나서 | `-고 나서` | 做完A之后B | 顺序动作 | 是 |

### 易错点

```korean
새집으로 이사하려면 가구를 새로 사야 한다.
```

中文：如果想搬到新房，就要重新买家具。

这个句子可以，但原题后面是 `샀다`，不是 `사야 한다`，所以不能选 `이사하려면`。

## 第3题

### 原题

```korean
어려운 이웃을 돕고자 매년 봉사 활동에 참여하고 있다.
```

题目要求找与 `돕고자` 意思最接近的表达。

### 正确答案

```korean
돕기 위해서
```

### 正确句子意思

中文：为了帮助困难的邻居，每年都参加志愿活动。

### 核心语法

| 语法 | 核心意思 | 本题意思 |
|---|---|---|
| `-고자` | 为了……、想要…… | 为了帮助 |
| `-기 위해서` | 为了…… | 为了帮助 |

### 同义关系

```text
돕고자 = 돕기 위해서
```

## 第4题

### 原题

```korean
지난 3년 동안 영화를 한 편 봤으니 거의 안 본 셈이다.
```

中文：过去三年只看了一部电影，所以几乎等于没看。

### 核心语法

```text
-(으)ㄴ 셈이다 = 算是…… / 等于…… / 可以说是……
```

### 选项语法整理

| 选项 | 语法 | 核心意思 | 中文 |
|---|---|---|---|
| 본 척했다 | `-(으)ㄴ 척하다` | 假装…… | 假装看过 |
| 보기 나름이다 | `-기 나름이다` | 取决于…… | 要看怎么看 |
| 볼 수밖에 없었다 | `-(으)ㄹ 수밖에 없다` | 不得不…… | 不得不看 |
| 안 본 거나 마찬가지이다 | `-거나 마찬가지이다` | 和……差不多 | 和没看差不多 |

### 重点对比

| 表达 | 中文 | 关系 |
|---|---|---|
| 거의 안 본 셈이다 | 几乎算是没看 | 原句 |
| 안 본 거나 마찬가지이다 | 和没看差不多 | 同义 |

### 注意

如果选项里没有 `안`，例如：

```korean
본 거나 마찬가지이다
```

意思是“和看过差不多”，和原句 `거의 안 본 셈이다` 意思相反。需要核对原卷或答案。

## 这4题必须掌握的语法

| 类别 | 语法 | 核心意思 | 例句 |
|---|---|---|---|
| 经历 | `-(으)ㄴ 적이 있다` | 有过……经历 | 한국에 가 본 적이 있다. |
| 尝试/经验 | `-아/어 보다` | 试着做/做过 | 한국 음식을 먹어 본 적이 있다. |
| 顺序 | `-고 나서` | 做完A之后B | 밥을 먹고 나서 공부했다. |
| 时间经过 | `-(으)ㄴ 지` | 做某事以来 | 한국어를 공부한 지 1년 됐다. |
| 条件 | `-거든` | 如果……的话 | 시간이 있거든 연락해. |
| 意图条件 | `-(으)려면` | 如果想…… | 합격하려면 공부해야 한다. |
| 目的 | `-고자` | 为了…… | 성공하고자 노력한다. |
| 目的 | `-기 위해서` | 为了…… | 한국어를 배우기 위해 공부한다. |
| 总结判断 | `-(으)ㄴ 셈이다` | 算是…… | 거의 안 본 셈이다. |
| 同等判断 | `-거나 마찬가지이다` | 和……差不多 | 안 한 거나 마찬가지이다. |
| 假装 | `-(으)ㄴ 척하다` | 假装…… | 모르는 척했다. |
| 取决于 | `-기 나름이다` | 取决于…… | 생각하기 나름이다. |
| 不得不 | `-(으)ㄹ 수밖에 없다` | 不得不…… | 기다릴 수밖에 없다. |
