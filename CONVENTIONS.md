# TOPIK4 Study Notes Conventions

生成日期：2026-06-17

## 1. 学习总原则

旧内容主动回忆
→ 旧内容检测
→ 今日新内容学习
→ 今日专项练习
→ 错题回炉
→ 睡前复习

核心规则：新词不等于必须主动输出。核心单词才要求韩语→中文、中文→韩语；固定搭配和句型块优先服务阅读理解。

## 2. 知识分类标准

| 类别 | 目录 | 判断标准 | 目标 |
|---|---|---|---|
| A 类：核心单词 Core Vocabulary | `vocabulary/` | 单个词、名词、动词、形容词、副词，适合主动背诵 | 韩语→中文、中文→韩语 |
| B 类：固定搭配 Phrases | `phrases/` | 两个以上词构成的常见搭配，不是完整句子 | 看到立即理解 |
| C 类：句型块 Chunks | `chunks/` | 较完整表达、半句、短句、阅读句型 | 阅读直接理解 |
| D 类：语法 Grammar | `grammar/` | 语法连接、句尾表达、助词结构、惯用语法 | TOPIK 题中稳定判断 |
| E 类：易混对比 Contrast | `contrast/` | vs、区别、容易混、区分、不是、A/B 类似项 | 快速排除混淆 |
| F 类：审校修正 Audit | `audit/` | 翻译错误、词义误记、长期容易背错内容 | 修正错因并回炉 |

## 3. 掌握等级标准

### 3.1 单词等级

| 等级 | 含义 |
|---|---|
| D | 完全陌生 |
| C | 认识但不会翻译 |
| B | 韩语→中文会 |
| A | 中文→韩语会 |
| S | 秒答 |

### 3.2 语法等级

| 等级 | 含义 |
|---|---|
| D | 完全不懂 |
| C | 知道中文意思 |
| B | 能区分相似语法 |
| A | TOPIK选择题稳定正确 |
| S | 阅读中自动理解 |

### 3.3 固定搭配和句型块等级

| 等级 | 含义 |
|---|---|
| D | 完全陌生 |
| C | 见过但不稳定 |
| B | 看到能理解 |
| A | 阅读中基本不卡 |
| S | 自动理解 |

## 4. R0-R7 复习周期

| 阶段 | 时间点 | 说明 |
|---|---|---|
| R0 | 学习后10分钟 | 当天即时回看 |
| R1 | 当天睡前 | 睡前主动回忆 |
| R2 | 第1天 | 第二天复习 |
| R3 | 第3天 | 第一次拉开间隔 |
| R4 | 第7天 | 周复习 |
| R5 | 第14天 | 两周复习 |
| R6 | 第30天 | 月复习 |
| R7 | 第60天 | 长间隔复习 |

回退规则：R3错回到R1；R7错回到R2；R5错回到R3；任何时候错都重新进入高频回炉。

## 5. 降频规则

| 条件 | 处理 |
|---|---|
| 连续正确2次 | 隔3天复习 |
| 连续正确3次 | 隔7天复习 |
| 连续正确5次 | 隔30天复习 |
| 连续正确8次 | 归档 |
| 归档后再错 | 回到高频回炉 |

## 6. 每日学习上限

| 项目 | 上限 |
|---|---:|
| 核心单词 | 15~25个 |

| 句型块 | 5个以内 |
| 语法 | 3~5个 |
| 阅读 | 10~20题 |
| 听力 | 10~20题 |

## 9. TOPIK 做题规则

做题目的：检测，不是学习。

不会的新知识，先记录；不立即加入复习卷；先进入学习区；讲解后再进入复习队列。

## 10. 禁止事项

- 只学新内容不复习
- 把所有内容都当作必须默写的单词
- 重复记录同一个知识点
- 删除历史文件
- 重命名历史文件
- 未确认 remote 就只在本地创建文件

## Daily Review Scheduling Rules

### 1. daily 是原始记录

daily/YYYY-MM-DD_daily_log.md 保存当天所有学习和复习痕迹，长期保留。

### 2. MASTER_REVIEW_QUEUE.md 是总调度表

所有复习内容必须进入总队列。

### 3. 每日 review plan 必须从总队列生成

review/YYYY-MM-DD_review_plan.md 不是手写清单，而是从 MASTER_REVIEW_QUEUE.md 中筛选所有到期内容。

筛选条件：

- 状态不是 archived
- 状态不是 pending
- 下次复习 <= 当天日期

### 4. 复习结果必须反向更新总队列

每天复习后，必须把 correct / wrong / uncertain 写回 daily，并通过脚本更新 MASTER_REVIEW_QUEUE.md。

### 5. 明天计划必须系统性生成

明天的复习计划必须包含所有到期内容，包括：

- 今天新学且到期的内容
- 以前学过且到期的内容
- 高频回炉内容
- 漏复习但仍到期的内容

禁止只根据当天新内容生成明天计划。
