# 2026-06-15 TOPIK 4级每日任务计划

> 制定日期：2026-06-15  
> 目标：7月或8月达到 TOPIK II 4级考试水平

---

## 1. 目标判断

TOPIK II 4级的核心不是单纯背语法，而是：

```text
阅读能稳定拿分 + 听力不崩 + 写作51~53题能拿基础分
```

TOPIK II 总分300分，4级目标线按 **150分以上** 准备。当前阶段应优先提高：

1. 阅读理解与选项判断
2. 听力关键词抓取
3. 高频单词和固定表达
4. 写作51、52、53题的基础得分能力

---

## 3. 推荐每日固定任务

### 3.1 阅读：每天10~15题

阅读不要全部精分，分成两类：

| 类型 | 数量 | 做法 |
|---|---:|---|
| 精分题 | 5~8题 | 逐句翻译、单词、语法、选项分析 |
| 限时题 | 5~7题 | 只练速度、关键词和答案判断 |

精分格式：

1. 题目问什么
2. 正确答案为什么对
3. 错误选项为什么错
4. 文章高频单词
5. 文章高频语法
6. 一句话中文总结

---

### 3.2 听力：每天10~15题

听力流程：

1. 第一遍正常做题
2. 对答案
3. 错题重听2遍
4. 把关键词写出来
5. 不做整篇听写，只精听3~5道错题

重点积累：

| 类型 | 例子 |
|---|---|
| 态度词 | 걱정하다, 만족하다, 아쉽다, 놀라다 |
| 行动词 | 신청하다, 예약하다, 취소하다, 확인하다 |
| 场景词 | 병원, 학교, 회사, 가게, 행사 |

---

### 3.3 单词：每天新词25~35个

优先背 TOPIK 题目中反复出现的词，不平均用力。

| 类型 | 例子 |
|---|---|
| 阅读高频抽象词 | 영향, 원인, 결과, 변화, 관심, 필요 |
| 新闻标题词 | 급감, 급증, 하락세, 상승세, 기지개 |
| 判断题关键词 | 적절하다, 알맞다, 일치하다, 설명하다 |
| 写作高频词 | 증가하다, 감소하다, 차이를 보이다, 영향을 미치다 |

整理规则：

```text
核心意思 → 常用意思 → 本句意思 → 固定搭配 → 例句
```

---

### 3.4 语法：每天3~5个

每个语法按固定格式整理：

```text
核心意思 + 本句意思 + 其他常见意思
```

近期优先语法：

| 语法 | 核心意思 |
|---|---|
| -기 위해서 | 为了…… |
| -에 따라 | 根据……；随着…… |
| -게 되다 | 变得……；最终…… |
| -도록 | 为了使……；到……程度 |
| -는 반면에 | 相反；另一方面 |

原则：不要只背某一句里的中文翻译，要理解语法功能。

---

### 3.5 写作：每天20~40分钟

写作不能等阅读和听力好了再开始。4级目标至少要保证51、52、53题能拿基础分。

| 题号 | 目标 |
|---|---|
| 51题 | 会填常见邮件、通知、请求句 |
| 52题 | 会根据上下文补句子 |
| 53题 | 会描述图表 |
| 54题 | 能写基本结构，不追求高级 |

每周安排：

| 星期 | 写作任务 |
|---|---|
| 周一 | 51、52各2题 |
| 周二 | 53图表1篇 |
| 周三 | 51、52各2题 |
| 周四 | 53图表1篇 |
| 周五 | 54大作文框架1篇 |
| 周六 | 51~53综合 |
| 周日 | 复盘本周写作模板 |
