# 混合间隔复习安排

目的：每天不只复习昨天内容，也要把前几天和高频错题混合进来，避免学完就忘。

---

## 1. 每日复习组成

每天开始的复习部分按这个比例出题：

| 来源 | 比例 | 内容 |
|---|---:|---|
| D1 | 40% | 昨天刚学内容 |
| D3 | 25% | 2到3天前内容 |
| D7 | 20% | 一周前内容 |
| 高频回炉 | 15% | 反复错、反应慢、影响做题的内容 |

如果当天复习量很小，也至少保证：

```text
D1 6题 + D3 4题 + 高频回炉 3题
```

---

## 2. 单词复习规则

单词复习时不按昨天原顺序排列，必须打乱顺序。

每次混合：

1. 昨天新词。
2. 前几天未稳词。
3. 高频错词。
4. 已掌握但需要低频抽查的词。

用户执行方式不变：

```text
看韩语写中文 → 遮住韩语按中文默写韩语 → 反馈不会、写错、反应慢
```

---

## 3. 语法复习规则

语法不按日期单独复习，要混合相近语法。

例如：

| 组别 | 混合内容 |
|---|---|
| 连接词 | 그래서 / 그래도 / 그러면 / 따라서 / 반면 / 하지만 |
| 同义替换 | 대로 / 것과 같이 / 탓에 / 바람에 |
| 可能推测 | ㄹ지도 모르다 / ㄹ 수도 있다 / ㄹ 리가 없다 |
| 时间顺序 | 고 나서 / ㄴ 지 / 기 전에 |

---

## 4. 题目生成规则

1. 题型可以重复。
2. 题干不能重复。
3. 选项顺序不能固定。
4. 主题要轮换，不能总是学习、发音、TOPIK。
5. D1之后不复用D0原题。

推荐主题池：

```text
饮食、交通、购物、打工、学校生活、健康、天气、旅行、公告、新闻、科技、生活服务、家庭、运动
```

---

## 5. 每日开头固定流程

```text
1. 混合旧词主动回忆
2. D1连接词或语法变式题
3. D3/D7旧语法混合题
4. 高频回炉题
5. 批改后再开始今天新内容
```

---

## 6. 降频规则

| 表现 | 处理 |
|---|---|
| 连续正确1次 | 继续D1/D3复习 |
| 连续正确2次 | 降到D7 |
| 连续正确3次 | 降到D14 |
| 连续正确5次 | 低频月复习 |
| 任何阶段再次错 | 拉回高频回炉 |

---

## 7. 6月8日开始执行

从 2026-06-08 起，复习卷不再只复习前一天，而是混合：

```text
6月7日内容 + 6月5日内容 + 高频错题
```

先复习，再学习新内容。
